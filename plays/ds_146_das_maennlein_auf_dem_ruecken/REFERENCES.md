# The Little Man on the Back: Reference

## Line of Work

Staging _Das Männlein auf dem Rücken_, legend no. 146 of the Brothers Grimm's _Deutsche Sagen_. A rope-maker kicks a boy's game board and is forced by the boy's father to carry him, dying ten days later.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 146.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 146 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
