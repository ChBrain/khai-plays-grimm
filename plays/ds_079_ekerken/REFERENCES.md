# Ekerken: Reference

## Line of Work

Staging _Ekerken_, legend no. 79 of the Brothers Grimm's _Deutsche Sagen_. A roadway spirit near Elten plagues travelers, remaining invisible except for a single human hand.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 79.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 79 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
