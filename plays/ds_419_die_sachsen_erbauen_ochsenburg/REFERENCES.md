# The Saxons Build Ochsenburg: Reference

## Line of Work

Staging _Die Sachsen erbauen Ochsenburg_, legend no. 419 of the Brothers Grimm's _Deutsche Sagen_. The Saxons cut an ox-hide into a thin cord, claiming a large territory to build Ochsenburg fortress.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 419.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 419 | The legend as the Grimms set it down. |

What the production refuses to model, and to whom it delegates.
