# The White Lady: Reference

## Line of Work

Staging _Die weiße Frau_, legend no. 123 of the Brothers Grimm's _Deutsche Sagen_. A white lady combs horses in a stable but is blind inside her dwelling and sees clearly only outside.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 123.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 123 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
