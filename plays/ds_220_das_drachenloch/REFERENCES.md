# The Dragon's Cave: Reference

## Line of Work

Staging _Das Drachenloch_, legend no. 220 of the Brothers Grimm's _Deutsche Sagen_. A count slays a giant dragon and retrieves his younger brother alive from the split belly of the monster.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 220.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 220 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
