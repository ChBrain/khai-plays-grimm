# The Emergency Shirt: Reference

## Line of Work

Staging _Das Noth-Hemd_, legend no. 255 of the Brothers Grimm's _Deutsche Sagen_. A magic linen shirt spun by two innocent young girls on Christmas Eve protects its wearer from all battle wounds.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 255.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 255 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
