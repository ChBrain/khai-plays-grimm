# The King's Grave: Reference

## Line of Work

Staging _Des Königs Grab_, legend no. 373 of the Brothers Grimm's _Deutsche Sagen_. The Goths divert a river to bury King Alaric with his treasures, executing the diggers to keep it secret.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 373.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 373 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
