# Charlemagne Besieges Pavia: Reference

## Line of Work

Staging _Carl belagert Pavia_, legend no. 448 of the Brothers Grimm's _Deutsche Sagen_. A princess betrays Pavia for Charlemagne's love, only to be trampled to death by the entering army.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 448.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 448 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
