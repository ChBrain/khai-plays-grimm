# The Faithless Stork: Reference

## Line of Work

Staging _Die treulose Störchin_, legend no. 498 of the Brothers Grimm's _Deutsche Sagen_. A male stork returns with other storks to judge and execute his unfaithful wife.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 498.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 498 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
