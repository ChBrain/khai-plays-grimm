# The Five Crosses: Reference

## Line of Work

Staging _Die fünf Kreuze_, legend no. 165 of the Brothers Grimm's _Deutsche Sagen_. Five ancient stone crosses standing outside the Klaus Gate in Höxter recall legends of giants and hung citizens.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 165.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 165 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
