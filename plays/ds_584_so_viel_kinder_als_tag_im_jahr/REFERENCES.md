# As Many Children as Days in the Year: Reference

## Line of Work

Staging _So viel Kinder, als Tag’ im Jahr_, legend no. 584 of the Brothers Grimm's _Deutsche Sagen_. Countess Margaretha mocks a poor beggar woman and is cursed to give birth to 365 children at once.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 584.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 584 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.
