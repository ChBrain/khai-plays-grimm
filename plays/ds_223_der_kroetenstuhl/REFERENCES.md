# The Toad Stool: Reference

## Line of Work

Staging _Der Krötenstuhl_, legend no. 223 of the Brothers Grimm's _Deutsche Sagen_. A proud cursed daughter appearing as a snake or toad must be kissed weekly on Nothweiler Castle to be redeemed.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 223.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 223 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
