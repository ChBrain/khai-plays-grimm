# The Lady of Bonikau: Reference

## Line of Work

Staging _Frau von Bonikau_, legend no. 70 of the Brothers Grimm's _Deutsche Sagen_. A lady in childbed allows earthmen to celebrate a wedding in her room, hosting an invisible festive assembly.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 70.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 70 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
