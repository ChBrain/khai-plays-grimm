# Play: Die Wiesenjungfrau

## Estate

[khai-plays-grimm](https://github.com/ChBrain/khai-plays-grimm): the repository that holds this production and answers for it.

## Name

Die Wiesenjungfrau.

## Arc

Ein Hirtenbube aus Auerbach sieht auf einer Thalwiese eine weiße Jungfrau, flieht aber vor Furcht. Später erscheint sie als Schlange mit einer blauen Blume im Mund, die der Schlüssel zu einer Schatzkammer ist. Der Bube läuft wieder davon. Beim dritten Mal bittet die Jungfrau flehentlich um Erlösung, doch er bekreuzigt sich nur. Die Jungfrau seufzt, dass sie nun warten muss, bis aus einer Kirschenbaumwiege ein Kind sie erlöst. Der Junge stirbt jung.
