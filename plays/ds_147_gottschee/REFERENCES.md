# Gottschee: Reference

## Line of Work

Staging _Gottschee_, legend no. 147 of the Brothers Grimm's _Deutsche Sagen_. A hunter is led into a subterranean cave by a spirit and witnesses the silent souls of former castle lords.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 147.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 147 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
