# The Judgment of Nobel: Reference

## Line of Work

Staging _The Judgment of Nobel_, an episode from Jacob Grimm's _Reinhart Fuchs_. Reinhart promises a hidden treasure to Nobel, receives a pardon, and leaves on a fake pilgrimage.

## Origin

Jacob Grimm's edition of the animal epic _Reinhart Fuchs_ (1834).

| Source          | Key Work / Event        | Scope                                           |
| --------------- | ----------------------- | ----------------------------------------------- |
| **Jacob Grimm** | _Reinhart Fuchs_ (1834) | The trickster episode as Jacob Grimm edited it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider epic**: the production stages this episode alone, not the full beast-epic it belongs to.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
