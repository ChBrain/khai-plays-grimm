# The Virgin of Oselberg: Reference

## Line of Work

Staging _Die Jungfrau im Oselberg_, legend no. 222 of the Brothers Grimm's _Deutsche Sagen_. A cursed castle maiden appears as a ghost snake with a human head and key ring during the quarterly Ember days.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 222.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 222 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
