# Gold Sand on the Unterberg: Reference

## Line of Work

Staging _Goldsand auf dem Unterberg_, legend no. 162 of the Brothers Grimm's _Deutsche Sagen_. Men discover gold sand on the Untersberg mountain but face warnings and a fleeting door into another world.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 162.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 162 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
