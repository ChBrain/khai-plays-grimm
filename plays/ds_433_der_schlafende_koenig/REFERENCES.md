# The Sleeping King: Reference

## Line of Work

Staging _Der schlafende König_, legend no. 433 of the Brothers Grimm's _Deutsche Sagen_. King Guntram's soul exits as a snake to find a treasure cave, guided by a servant's sword bridge.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 433.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 433 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
