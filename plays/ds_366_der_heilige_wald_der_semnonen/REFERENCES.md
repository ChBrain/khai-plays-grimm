# The Sacred Forest of the Semnones: Reference

## Line of Work

Staging _Der heilige Wald der Semnonen_, legend no. 366 of the Brothers Grimm's _Deutsche Sagen_. Semnones perform a human sacrifice in a sacred grove where everyone must enter bound in chains of submission.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 366.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 366 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
