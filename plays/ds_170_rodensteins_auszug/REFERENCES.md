# Rodenstein's Departure: Reference

## Line of Work

Staging _Rodensteins Auszug_, legend no. 170 of the Brothers Grimm's _Deutsche Sagen_. The ghost of the Lord of Rodenstein noisily marches to Schnellerts castle when war looms and returns in silence.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 170.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 170 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
