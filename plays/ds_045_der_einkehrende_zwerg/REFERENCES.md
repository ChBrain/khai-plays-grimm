# The Lodging Dwarf: Reference

## Line of Work

Staging _Der einkehrende Zwerg_, legend no. 45 of the Brothers Grimm's _Deutsche Sagen_. A wet dwarf seeks shelter in a storm; only a poor pious couple takes him in, saving them from a devastating landslide.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 45.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 45 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
