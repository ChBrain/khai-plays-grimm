# The Triumph of the Trickster: Reference

## Line of Work

Staging _The Triumph of the Trickster_, an episode from Jacob Grimm's _Reinhart Fuchs_. Reinhart returns to the court as a false physician, poisons the Lion-King to death, and flees back to his fortress.

## Origin

Jacob Grimm's edition of the animal epic _Reinhart Fuchs_ (1834).

| Source          | Key Work / Event        | Scope                                           |
| --------------- | ----------------------- | ----------------------------------------------- |
| **Jacob Grimm** | _Reinhart Fuchs_ (1834) | The trickster episode as Jacob Grimm edited it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider epic**: the production stages this episode alone, not the full beast-epic it belongs to.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
