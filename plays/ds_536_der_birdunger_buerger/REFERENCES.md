# The Verdun Citizen: Reference

## Line of Work

Staging _Der Birdunger Bürger_, legend no. 536 of the Brothers Grimm's _Deutsche Sagen_. A Verdun citizen makes a pact with the devil for money and is eventually carried off to hell on a black steed.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 536.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 536 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
