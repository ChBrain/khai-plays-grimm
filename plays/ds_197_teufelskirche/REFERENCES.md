# The Devil's Church: Reference

## Line of Work

Staging _Teufelskirche_, legend no. 197 of the Brothers Grimm's _Deutsche Sagen_. The devil steals building stones from a valley church project and piles them up on a Rhön mountain peak.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 197.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 197 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
