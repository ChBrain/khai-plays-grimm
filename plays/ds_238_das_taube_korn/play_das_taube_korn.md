# Play: Das taube Korn

## Estate

[khai-plays-grimm](https://github.com/ChBrain/khai-plays-grimm): the repository that holds this production and answers for it.

## Name

Das taube Korn.

## Arc

Die Einwohner von Stavoren in Friesland werden durch unermesslichen Reichtum stolz. Eine alte geizige Witwe beauftragt einen Danzigfahrer, ihr das edelste Gut auf Erden zu bringen. Er lädt Weizen; da die Preise verfallen sind, lässt die Witwe die Ladung zornig im Hafen ins Meer werfen. Als Strafe wächst dort eine Sandbank namens Frauensand heran, auf der nur nutzloses, taubes Korn gedeiht und den Hafen ruiniert.
