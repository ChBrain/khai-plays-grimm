# The Gushing Fountain: Reference

## Line of Work

Staging _Der quillende Brunnen_, legend no. 104 of the Brothers Grimm's _Deutsche Sagen_. A fountain dries up whenever a member of a noble Franconian house is dying, but the omen is cheated.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 104.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 104 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
