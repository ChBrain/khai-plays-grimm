# Sinful Love: Reference

## Line of Work

Staging _Sündliche Liebe_, legend no. 358 of the Brothers Grimm's _Deutsche Sagen_. A brother and sister commit incest and are decapitated by the devil, leaving a headless mark on their gravestone.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 358.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 358 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
