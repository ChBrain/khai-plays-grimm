# The Wandering Hunter: Reference

## Line of Work

Staging _Der herumziehende Jäger_, legend no. 258 of the Brothers Grimm's _Deutsche Sagen_. A mysterious hunter puts an end to a series of phantom sniper killings in a haunted forest.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 258.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 258 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
