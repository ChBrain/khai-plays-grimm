# Wittekind's Flight: Reference

## Line of Work

Staging _Wittekinds Flucht_, legend no. 454 of the Brothers Grimm's _Deutsche Sagen_. Defeated Saxons bury an old woman alive at a ford to prevent her capture, leaving a haunted field.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 454.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 454 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
