# Eginhard and Emma: Reference

## Line of Work

Staging _Eginhart und Emma_, legend no. 457 of the Brothers Grimm's _Deutsche Sagen_. Emma carries Eginhard across the snowy courtyard to hide footprints; Charlemagne witnesses it and blesses them.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 457.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 457 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
