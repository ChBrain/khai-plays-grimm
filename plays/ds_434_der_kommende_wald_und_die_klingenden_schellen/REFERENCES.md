# The Moving Forest and the Ringing Bells: Reference

## Line of Work

Staging _Der kommende Wald und die klingenden Schellen_, legend no. 434 of the Brothers Grimm's _Deutsche Sagen_. Frankish soldiers carry tree branches and horse bells to trick Childebert's army into routing.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 434.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 434 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
