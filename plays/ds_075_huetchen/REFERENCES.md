# Little Hat: Reference

## Line of Work

Staging _Hütchen_, legend no. 75 of the Brothers Grimm's _Deutsche Sagen_. A house spirit wearing a small felt hat serves the bishop of Hildesheim, but takes bloody revenge when mocked.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 75.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 75 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
