# The Ghostly Banquet: Reference

## Line of Work

Staging _Geister-Mahl_, legend no. 177 of the Brothers Grimm's _Deutsche Sagen_. A traveler in a haunted room witnesses a ghostly feast that vanishes upon prayer, leaving behind valuable silver.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 177.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 177 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
