# The Huelfenberg: Reference

## Line of Work

Staging _Der Hülfenberg_, legend no. 182 of the Brothers Grimm's _Deutsche Sagen_. A tricked devil bursts out of a chapel on the Hülfenberg, leaving a hole in the gable that can never be closed.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 182.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 182 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
