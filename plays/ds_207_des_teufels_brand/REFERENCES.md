# The Devil's Fire: Reference

## Line of Work

Staging _Des Teufels Brand_, legend no. 207 of the Brothers Grimm's _Deutsche Sagen_. The devil mockingly exposes the sins of local priests and burns down the town of Schiltach using a magic kettle.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 207.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 207 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
