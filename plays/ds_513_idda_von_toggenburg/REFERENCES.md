# Idda of Toggenburg: Reference

## Line of Work

Staging _Idda von Toggenburg_, legend no. 513 of the Brothers Grimm's _Deutsche Sagen_. Accused of adultery due to a stolen ring, Idda survives a fall from the cliff and lives as a holy hermit.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 513.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 513 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
