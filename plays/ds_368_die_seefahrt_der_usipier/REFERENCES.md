# The Sea Voyage of the Usipii: Reference

## Line of Work

Staging _Die Seefahrt der Usipier_, legend no. 368 of the Brothers Grimm's _Deutsche Sagen_. Mutinous Germanic conscripts flee Britain in stolen ships, resort to cannibalism, and are sold as slaves.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 368.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 368 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
