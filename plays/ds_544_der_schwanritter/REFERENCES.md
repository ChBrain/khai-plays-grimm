# The Swan Knight: Reference

## Line of Work

Staging _Der Schwanritter_, legend no. 544 of the Brothers Grimm's _Deutsche Sagen_. The Duke of Saxony seizes Brabant; a swan knight defends the Duchess's daughter but leaves when questioned.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 544.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 544 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
