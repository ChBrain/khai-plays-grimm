# The Count of Hoia: Reference

## Line of Work

Staging _Der Graf von Hoia_, legend no. 35 of the Brothers Grimm's _Deutsche Sagen_. A dwarf asks the Count of Hoia to host a nightly feast in his castle, rewarding him with a sword and a ring.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 35.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 35 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
