# Hermann of Rosenberg: Reference

## Line of Work

Staging _Hermann von Rosenberg_, legend no. 42 of the Brothers Grimm's _Deutsche Sagen_. After Hermann von Rosenberg's feast, earth spirits request to hold their own wedding in his hall, promising peace.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 42.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 42 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
