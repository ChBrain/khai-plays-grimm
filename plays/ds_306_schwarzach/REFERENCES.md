# Schwarzach: Reference

## Line of Work

Staging _Schwarzach_, legend no. 306 of the Brothers Grimm's _Deutsche Sagen_. A knight tastes a forbidden water drink and sinks into a lake where his daughter is held captive by a giant snake.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 306.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 306 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
