# The Giant Haym: Reference

## Line of Work

Staging _Riese Haym_, legend no. 140 of the Brothers Grimm's _Deutsche Sagen_. The giant Haym slays a dragon in the Inntal valley, builds Wilten Abbey, and erects the bridge at Innsbruck.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 140.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 140 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
