# Grinken-Smith: Reference

## Line of Work

Staging _Grinken-Schmidt_, legend no. 157 of the Brothers Grimm's _Deutsche Sagen_. A legendary subterranean smith kills a farmer's horse after the farmer refuses to pay him for borrowing a spit.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 157.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 157 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
