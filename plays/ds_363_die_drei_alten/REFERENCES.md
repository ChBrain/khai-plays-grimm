# The Three Old Men: Reference

## Line of Work

Staging _Die drei Alten_, legend no. 363 of the Brothers Grimm's _Deutsche Sagen_. A priest is shown three extremely ancient men, each older than the last, who live in a state of holy waiting.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 363.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 363 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
