# Giants from the Untersberg: Reference

## Line of Work

Staging _Riesen aus dem Untersberge_, legend no. 138 of the Brothers Grimm's _Deutsche Sagen_. Giants descend from the Untersberg mountain to warn Feldkirchen villagers to live a righteous life.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 138.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 138 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
