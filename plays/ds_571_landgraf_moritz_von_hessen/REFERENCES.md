# Landgrave Maurice of Hesse: Reference

## Line of Work

Staging _Landgraf Moritz von Hessen_, legend no. 571 of the Brothers Grimm's _Deutsche Sagen_. Moritz disguises as a beggar to join a wealthy soldier in a nightly treasure hunt using magic roots.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 571.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 571 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
