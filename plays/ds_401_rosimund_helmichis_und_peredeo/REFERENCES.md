# Rosimund, Helmichis, and Peredeo: Reference

## Line of Work

Staging _Rosimund, Helmichis und Peredeo_, legend no. 401 of the Brothers Grimm's _Deutsche Sagen_. After Alboin's murder, Helmichis and Rosimund flee to Ravenna, where they both end up poisoning each other.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 401.

| Source             | Key Work / Event         | Scope                                 |

| **Brothers Grimm** | _Deutsche Sagen_, DS 401 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
