# The Werewolf Stone: Reference

## Line of Work

Staging _Der Wärwolf-Stein_, legend no. 215 of the Brothers Grimm's _Deutsche Sagen_. A shepherd chases a mocking werewolf assistant who transforms into a thornbush before being struck down.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 215.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 215 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
