# Legend of the Student Hildebrand: Reference

## Line of Work

Staging _Sage vom Schüler Hildebrand_, legend no. 490 of the Brothers Grimm's _Deutsche Sagen_. Young Hildebrand's papacy is predicted; Prince Henry IV hates the boy out of instinctive future rivalry.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 490.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 490 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
