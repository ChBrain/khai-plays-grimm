# Migration of the Goths: Reference

## Line of Work

Staging _Wanderung der Gothen_, legend no. 369 of the Brothers Grimm's _Deutsche Sagen_. The Goths migrate under their first kings, but a collapsed bridge permanently divides their army during migration.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 369.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 369 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
