# Mother Holle's Pond: Reference

## Line of Work

Staging _Frau Hollen Teich_, legend no. 4 of the Brothers Grimm's _Deutsche Sagen_. The pond on Hoher Meißner acts as the gateway to Mother Holle's underworld castle where she rewards the industrious.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 4.

| Source             | Key Work / Event       | Scope                                 |
| ------------------ | ---------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 4 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
