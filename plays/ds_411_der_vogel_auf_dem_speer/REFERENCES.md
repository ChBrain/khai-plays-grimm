# The Bird on the Spear: Reference

## Line of Work

Staging _Der Vogel auf dem Speer_, legend no. 411 of the Brothers Grimm's _Deutsche Sagen_. A cuckoo perching on Hildeprand's coronation spear is interpreted as a sign of a useless and brief reign.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 411.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 411 | The legend as the Grimms set it down. |

What the production refuses to model, and to whom it delegates.
