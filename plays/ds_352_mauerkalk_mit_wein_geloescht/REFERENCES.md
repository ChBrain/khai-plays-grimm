# Mortar Slaked with Wine: Reference

## Line of Work

Staging _Mauerkalk mit Wein gelöscht_, legend no. 352 of the Brothers Grimm's _Deutsche Sagen_. Emperor Friedrich the Third orders citizens to use extremely sour wine to slake mortar for building Saint Stephen.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 352.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 352 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
