# Founding of Wettenhausen Monastery: Reference

## Line of Work

Staging _Stiftung des Klosters Wettenhausen_, legend no. 532 of the Brothers Grimm's _Deutsche Sagen_. Countess Gertrud uses a tiny plow hidden in her bosom and a horse ride to claim land for a monastery.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 532.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 532 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
