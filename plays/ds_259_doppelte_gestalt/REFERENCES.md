# Double Form: Reference

## Line of Work

Staging _Doppelte Gestalt_, legend no. 259 of the Brothers Grimm's _Deutsche Sagen_. A sick nobleman is tricked by a traveler into accusing his honest neighbor of witchcraft through a phantom double.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 259.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 259 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
