# The Dripping Rib: Reference

## Line of Work

Staging _Die tropfende Rippe_, legend no. 141 of the Brothers Grimm's _Deutsche Sagen_. A giant pagan maiden's rib drips once a year in Oberburg church, with the final drop signaling Judgment Day.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 141.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 141 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
