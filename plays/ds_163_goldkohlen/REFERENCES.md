# Gold Coals: Reference

## Line of Work

Staging _Gold-Kohlen_, legend no. 163 of the Brothers Grimm's _Deutsche Sagen_. An herb gatherer on the Untersberg mountain finds gold-rich gray coals but can never rediscover the stone wall.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 163.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 163 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
