# The Three Treasure Diggers: Reference

## Line of Work

Staging _Die drei Schatzgräber_, legend no. 334 of the Brothers Grimm's _Deutsche Sagen_. Three craftsmen attempting to dig up a church treasure near Erfurt are killed by the guarding evil spirit.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 334.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 334 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
