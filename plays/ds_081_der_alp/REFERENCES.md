# The Nightmare: Reference

## Line of Work

Staging _Der Alp_, legend no. 81 of the Brothers Grimm's _Deutsche Sagen_. The nightmare spirit slips through tiny cracks to press on sleepers' chests, fleeing only if one moves a thumb.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 81.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 81 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
