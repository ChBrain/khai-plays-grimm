# The Oldenburg Horn: Reference

## Line of Work

Staging _Das oldenburger Horn_, legend no. 547 of the Brothers Grimm's _Deutsche Sagen_. Otto of Oldenburg is offered a drink by a mountain maiden, but pours it away and steals the horn.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 547.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 547 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
