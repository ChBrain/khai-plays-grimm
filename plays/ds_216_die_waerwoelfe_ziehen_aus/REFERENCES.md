# The Werewolves Set Out: Reference

## Line of Work

Staging _Die Wärwölfe ziehen aus_, legend no. 216 of the Brothers Grimm's _Deutsche Sagen_. A leader with a wire whip drives a horde of shape-shifted werewolves across parted waters to raid Livonian herds.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 216.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 216 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
