# Brother Nickel: Reference

## Line of Work

Staging _Bruder Nickel_, legend no. 55 of the Brothers Grimm's _Deutsche Sagen_. Fishermen on Rügen find their boat placed high in a tree by a hidden water spirit named Brother Nickel.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 55.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 55 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
