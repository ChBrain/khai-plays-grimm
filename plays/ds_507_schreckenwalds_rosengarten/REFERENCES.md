# Schreckenwald's Rose Garden: Reference

## Line of Work

Staging _Schreckenwalds Rosengarten_, legend no. 507 of the Brothers Grimm's _Deutsche Sagen_. Robber Schreckenwald imprisons victims in a tiny cliff ledge; one escapes via a daring jump, sealing the robber's doom.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 507.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 507 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
