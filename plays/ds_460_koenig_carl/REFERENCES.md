# King Charles: Reference

## Line of Work

Staging _König Carl_, legend no. 460 of the Brothers Grimm's _Deutsche Sagen_. Young Charles is summoned to Rome by a dream voice, where his brother Pope Leo crowns him Emperor.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 460.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 460 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
