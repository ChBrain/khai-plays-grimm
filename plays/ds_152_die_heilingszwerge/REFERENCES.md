# The Heiling Dwarves: Reference

## Line of Work

Staging _Die Heilingszwerge_, legend no. 152 of the Brothers Grimm's _Deutsche Sagen_. A berry-picking woman enters the dwarf-prince Heiling's cave and awakens a century later to find her world gone.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 152.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 152 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
