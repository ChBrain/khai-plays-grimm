# Slit-Ears: Reference

## Line of Work

Staging _Schlitzöhrchen_, legend no. 63 of the Brothers Grimm's _Deutsche Sagen_. A malicious water spirit named Slit-Ears lurks in the river Streu to drag crossing travelers to their deaths.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 63.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 63 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
