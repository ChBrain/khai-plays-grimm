# The Merovingians: Reference

## Line of Work

Staging _Die Merovinger_, legend no. 424 of the Brothers Grimm's _Deutsche Sagen_. Queen Clodio is overwhelmed by a sea monster, giving birth to Merovech, the ancestor of the bristly kings.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 424.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 424 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
