# Hans Heiling's Rocks: Reference

## Line of Work

Staging _Hans Heilings Felsen_, legend no. 329 of the Brothers Grimm's _Deutsche Sagen_. Hans Heiling makes a pact with the devil to turn a wedding party to stone after the bride rejects him.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 329.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 329 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
