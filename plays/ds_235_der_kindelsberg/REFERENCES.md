# The Kindelsberg: Reference

## Line of Work

Staging _Der Kindelsberg_, legend no. 235 of the Brothers Grimm's _Deutsche Sagen_. Overproud knights are struck by the plague before a beautiful daughter is slain by a rejected black knight.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 235.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 235 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
