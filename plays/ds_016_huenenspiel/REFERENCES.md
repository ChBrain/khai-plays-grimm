# Giants' Game: Reference

## Line of Work

Staging _Hünenspiel_, legend no. 16 of the Brothers Grimm's _Deutsche Sagen_. Giants playing on the Brunsberg throw a ball into the valley, forming the Knäuelwiese before their demise.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 16.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 16 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
