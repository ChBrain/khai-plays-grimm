# Oxen Show the Holy Site: Reference

## Line of Work

Staging _Ochsen zeigen die heilige Stätte_, legend no. 350 of the Brothers Grimm's _Deutsche Sagen_. A community uses yoked oxen to locate the divinely chosen site for building Saint Stephen church.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 350.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 350 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
