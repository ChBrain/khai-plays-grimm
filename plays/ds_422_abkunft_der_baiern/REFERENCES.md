# Descent of the Bavarians: Reference

## Line of Work

Staging _Abkunft der Baiern_, legend no. 422 of the Brothers Grimm's _Deutsche Sagen_. The Bavarians immigrate from Armenia, forging legendary swords and fighting Julius Caesar.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 422.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 422 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
