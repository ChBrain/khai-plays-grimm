# The Brennberger, First Legend: Reference

## Line of Work

Staging _Der Brennberger, erste Sage_, legend no. 505 of the Brothers Grimm's _Deutsche Sagen_. The Duchess of Austria sends the singing knight Brennberger to France to compare her beauty to the Queen's.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 505.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 505 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
