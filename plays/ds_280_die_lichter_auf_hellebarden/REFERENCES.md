# The Lights on Halberds: Reference

## Line of Work

Staging _Die Lichter auf Hellebarden_, legend no. 280 of the Brothers Grimm's _Deutsche Sagen_. Brushing a warning light off a pitchfork three times results in a fatal duel between two close farmer friends.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 280.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 280 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
