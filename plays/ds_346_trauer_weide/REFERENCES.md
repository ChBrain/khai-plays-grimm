# The Weeping Willow: Reference

## Line of Work

Staging _Trauer-Weide_, legend no. 346 of the Brothers Grimm's _Deutsche Sagen_. The weeping willow hangs its branches to the earth in eternal mourning for being used to scourge Christ.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 346.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 346 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
