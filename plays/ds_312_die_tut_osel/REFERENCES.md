# The Screech Owl: Reference

## Line of Work

Staging _Die Tut-Osel_, legend no. 312 of the Brothers Grimm's _Deutsche Sagen_. A howling Thuringian nun is banished as an owl to the Harz where she joins the wild hunt of Hackelnberg.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 312.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 312 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
