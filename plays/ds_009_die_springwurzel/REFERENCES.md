# The Springing Root: Reference

## Line of Work

Staging _Die Springwurzel_, legend no. 9 of the Brothers Grimm's _Deutsche Sagen_. A shepherd uses a magical spring-root to open the locked doors inside the Köterberg and gather treasures.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 9.

| Source             | Key Work / Event       | Scope                                 |
| ------------------ | ---------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 9 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
