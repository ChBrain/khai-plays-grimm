# The Devil's Wall: Reference

## Line of Work

Staging _Die Teufelsmauer_, legend no. 189 of the Brothers Grimm's _Deutsche Sagen_. A premature rooster crow ruins a pact between God and the devil, who breaks his own wall in a furious rage.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 189.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 189 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
