# Childeric and Basina: Reference

## Line of Work

Staging _Childerich und Basina_, legend no. 425 of the Brothers Grimm's _Deutsche Sagen_. An exiled King uses a split gold ring to signal his return, winning the Thuringian Queen Basina.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 425.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 425 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
