# The Cursed Maiden: Reference

## Line of Work

Staging _Die verfluchte Jungfer_, legend no. 227 of the Brothers Grimm's _Deutsche Sagen_. A stubborn daughter cursed to dwell in an Eisenach cave can only be saved if a traveler blesses her when she sneezes.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 227.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 227 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
