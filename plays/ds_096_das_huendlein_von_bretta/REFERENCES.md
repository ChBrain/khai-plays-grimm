# The Little Dog of Bretta: Reference

## Line of Work

Staging _Das Hündlein von Bretta_, legend no. 96 of the Brothers Grimm's _Deutsche Sagen_. A loyal dog sent to a butcher with a sausage request on a fast day has its tail chopped off in retaliation.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 96.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 96 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
