# The Miraculous Image from the Larch Trunk at Waldrast: Reference

## Line of Work

Staging _Das Gnadenbild aus dem Lerchenstock zu Waldrast_, legend no. 349 of the Brothers Grimm's _Deutsche Sagen_. An angel commands a larch trunk to grow an image of Mary, and a woodcutter is chosen by a voice to build a chapel.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 349.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 349 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
