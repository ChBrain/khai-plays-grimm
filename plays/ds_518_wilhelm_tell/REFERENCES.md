# William Tell: Reference

## Line of Work

Staging _Wilhelm Tell_, legend no. 518 of the Brothers Grimm's _Deutsche Sagen_. Tell refuses to bow to Gessler's hat, shoots an apple off his son's head, and kills the tyrant.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 518.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 518 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
