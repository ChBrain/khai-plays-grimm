# The War Sword: Reference

## Line of Work

Staging _Das Kriegsschwert_, legend no. 381 of the Brothers Grimm's _Deutsche Sagen_. A shepherd finds a buried sword that wounded a cow and presents it to King Attila as a sign of world dominion.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 381.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 381 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
