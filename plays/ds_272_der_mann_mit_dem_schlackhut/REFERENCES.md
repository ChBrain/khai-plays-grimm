# The Man with the Slouch Hat: Reference

## Line of Work

Staging _Der Mann mit dem Schlackhut_, legend no. 272 of the Brothers Grimm's _Deutsche Sagen_. A bearded ghost in a slouch hat visits an old woman in a ruined castle and silently smokes a pipe at her table.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 272.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 272 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.
