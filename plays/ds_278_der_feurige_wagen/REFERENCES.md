# The Fiery Carriage: Reference

## Line of Work

Staging _Der feurige Wagen_, legend no. 278 of the Brothers Grimm's _Deutsche Sagen_. A watchman in Odenwald encounters two horrific fiery carriages carrying ghosts with flames shooting from their mouths.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 278.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 278 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
