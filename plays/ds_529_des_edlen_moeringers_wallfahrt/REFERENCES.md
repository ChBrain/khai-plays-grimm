# The Noble Moringer's Pilgrimage: Reference

## Line of Work

Staging _Des edlen Möringers Wallfahrt_, legend no. 529 of the Brothers Grimm's _Deutsche Sagen_. Möringer returns from a 7-year pilgrimage just in time to stop his wife's wedding to Neufen.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 529.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 529 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
