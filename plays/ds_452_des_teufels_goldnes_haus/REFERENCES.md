# The Devil's Golden House: Reference

## Line of Work

Staging _Des Teufels goldnes Haus_, legend no. 452 of the Brothers Grimm's _Deutsche Sagen_. Radbod's messenger is guided by an angel to expose the devil's magnificent golden palace as a sulfur pit.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 452.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 452 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
