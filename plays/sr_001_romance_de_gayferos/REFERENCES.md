# Gayferos' Lay: Reference

## Line of Work

Staging _Gayferos' Lay_, a romance from Jacob Grimm's _Silva de romances viejos_. Don Gayferos is rebuked for playing dice instead of rescuing his captive wife Melisenda, whom he then saves.

## Origin

Jacob Grimm's anthology _Silva de romances viejos_ (1815).

| Source          | Key Work / Event                  | Scope                                               |
| --------------- | --------------------------------- | --------------------------------------------------- |
| **Jacob Grimm** | _Silva de romances viejos_ (1815) | The Spanish romance as Jacob Grimm anthologised it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider balladry**: the production stages this romance alone, not the tradition it is drawn from.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
