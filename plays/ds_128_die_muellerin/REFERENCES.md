# The Miller's Wife: Reference

## Line of Work

Staging _Die Müllerin_, legend no. 128 of the Brothers Grimm's _Deutsche Sagen_. A miller's wife traps one robber in a chamber and kills another in the waterwheel while her husband is away at church.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 128.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 128 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
