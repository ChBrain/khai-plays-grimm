# The Lombard Minstrel: Reference

## Line of Work

Staging _Der lombardische Spielmann_, legend no. 446 of the Brothers Grimm's _Deutsche Sagen_. A minstrel sings of a secret pass and guides Charlemagne's army over the mountains to defeat the Lombards.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 446.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 446 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
