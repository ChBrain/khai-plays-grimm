# The Kobold in the Mill: Reference

## Line of Work

Staging _Der Kobold in der Mühle_, legend no. 74 of the Brothers Grimm's _Deutsche Sagen_. Two students shelter in a mill where one eats the food set out for the house spirit, earning a midnight thrashing.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 74.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 74 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
