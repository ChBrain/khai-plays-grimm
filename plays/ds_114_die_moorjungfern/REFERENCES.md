# The Maidens of the Moor: Reference

## Line of Work

Staging _Die Moorjungfern_, legend no. 114 of the Brothers Grimm's _Deutsche Sagen_. Ghostly lights hover over a sunken village in the Red Moor, while a stone bridge remains in the Black Moor.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 114.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 114 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
