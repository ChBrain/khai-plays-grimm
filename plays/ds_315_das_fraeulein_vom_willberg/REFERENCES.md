# The Lady of Willberg: Reference

## Line of Work

Staging _Das Fräulein vom Willberg_, legend no. 315 of the Brothers Grimm's _Deutsche Sagen_. A man uses a magic bouquet to enter a mountain cave of gold but forgets it, turning his treasures to paper.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 315.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 315 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
