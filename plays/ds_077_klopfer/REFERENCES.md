# Knocker: Reference

## Line of Work

Staging _Klopfer_, legend no. 77 of the Brothers Grimm's _Deutsche Sagen_. A helpful household spirit in Flügelau Castle flees in a trail of fire when forced to reveal his true form.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 77.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 77 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
