# The Snake Maiden: Reference

## Line of Work

Staging _Die Schlangenjungfrau_, legend no. 13 of the Brothers Grimm's _Deutsche Sagen_. Leonhard enters a subterranean vault where a half-snake maiden promises treasures, but he flees in terror.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 13.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 13 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
