# Giant Stones: Reference

## Line of Work

Staging _Riesensteine_, legend no. 135 of the Brothers Grimm's _Deutsche Sagen_. Giant stones with handprints are found near Leipzig and Salzwedel, where a commander once split a rock with his sword.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 135.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 135 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
