# Crystal Gazing: Reference

## Line of Work

Staging _Krystall-Schauen_, legend no. 119 of the Brothers Grimm's _Deutsche Sagen_. A girl uses crystal gazing to discover her future husband, with Johann Rüst witnessing the magical vision.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 119.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 119 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
