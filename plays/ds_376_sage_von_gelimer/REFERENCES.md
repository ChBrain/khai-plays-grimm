# Saga of Gelimer: Reference

## Line of Work

Staging _Sage von Gelimer_, legend no. 376 of the Brothers Grimm's _Deutsche Sagen_. An accidental arrow hit from a drunk bodyguard kills a commander, allowing Vandal King Gelimer to escape.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 376.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 376 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
