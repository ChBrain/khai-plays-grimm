# The Devil's Pool near Dassel: Reference

## Line of Work

Staging _Das Teufelsbad bei Dassel_, legend no. 203 of the Brothers Grimm's _Deutsche Sagen_. The devil steals a golden bell and later drags an impious plowing farmer down into the bottomless Teufelsbad.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 203.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 203 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.
