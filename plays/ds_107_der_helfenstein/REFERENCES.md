# The Helfenstein: Reference

## Line of Work

Staging _Der Helfenstein_, legend no. 107 of the Brothers Grimm's _Deutsche Sagen_. A mountain door opens in 1614 for a Marschendorf maid, showing a mystical hall of zeching lords.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 107.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 107 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
