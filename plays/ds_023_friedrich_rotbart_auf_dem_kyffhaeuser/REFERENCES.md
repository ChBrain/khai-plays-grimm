# Barbarossa in the Kyffhaeuser: Reference

## Line of Work

Staging _Friedrich Rotbart auf dem Kyffhäuser_, legend no. 23 of the Brothers Grimm's _Deutsche Sagen_. Emperor Frederick Barbarossa sleeps inside the Kyffhäuser, awaiting the day the ravens stop flying to return.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 23.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 23 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
