# Origin of the Franks: Reference

## Line of Work

Staging _Herkunft der Franken_, legend no. 423 of the Brothers Grimm's _Deutsche Sagen_. Trojans escaping the fall of Troy wander to the Rhine, founding a new Troy called Santen.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 423.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 423 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
