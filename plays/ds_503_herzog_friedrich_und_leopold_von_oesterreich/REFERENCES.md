# Duke Frederick and Leopold of Austria: Reference

## Line of Work

Staging _Herzog Friedrich und Leopold von Oesterreich_, legend no. 503 of the Brothers Grimm's _Deutsche Sagen_. Leopold summons the Devil to free King Frederick, but Frederick refuses to fly with the Devil.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 503.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 503 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
