# The Footprint of the Lord: Reference

## Line of Work

Staging _Der Herrgottstritt_, legend no. 185 of the Brothers Grimm's _Deutsche Sagen_. Christ leaves footprint imprints in rock after banishing the devil to a ravine to weep tears for a thousand years.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 185.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 185 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
