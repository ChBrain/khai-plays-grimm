# The Children of Hameln: Reference

## Line of Work

Staging _Die Kinder zu Hameln_, legend no. 245 of the Brothers Grimm's _Deutsche Sagen_. A strange piper lures Hamelin's rats into a river and later leads all the children away when citizens refuse pay.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 245.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 245 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
