# The Moss Woman: Reference

## Line of Work

Staging _Das Moosweibchen_, legend no. 47 of the Brothers Grimm's _Deutsche Sagen_. A woodcutter Hans Krepel refuses a moss woman's plea to carve crosses in the last tree stump, triggering her wrath.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 47.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 47 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
