# Origin of the Huns: Reference

## Line of Work

Staging _Ursprung der Hunnen_, legend no. 378 of the Brothers Grimm's _Deutsche Sagen_. Banished Gothic witches mate with wild forest-spirits in the desert, giving rise to the feral Huns.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 378.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 378 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
