# The Shirt Casting: Reference

## Line of Work

Staging _Das Hemdabwerfen_, legend no. 118 of the Brothers Grimm's _Deutsche Sagen_. A girl uses a Christmas shirt magic ritual to see her future husband, but a group of girls confuses the spirits.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 118.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 118 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
