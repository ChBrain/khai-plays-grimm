# The Counts of Eberstein: Reference

## Line of Work

Staging _Die Grafen von Eberstein_, legend no. 476 of the Brothers Grimm's _Deutsche Sagen_. An Emperor's daughter warns a Count of a tournament plot to seize his castle, leading to peace and marriage.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 476.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 476 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
