# The Witches' Dance: Reference

## Line of Work

Staging _Der Hexen-Tanz_, legend no. 252 of the Brothers Grimm's _Deutsche Sagen_. A whistling boy who climbs a tree to play for a witch gathering invokes God's name, causing them all to vanish.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 252.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 252 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
