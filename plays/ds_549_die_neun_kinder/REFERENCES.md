# The Nine Children: Reference

## Line of Work

Staging _Die neun Kinder_, legend no. 549 of the Brothers Grimm's _Deutsche Sagen_. Uffo dreams his wife bore him nine children, which turn out to be nine churches she founded.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 549.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 549 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
