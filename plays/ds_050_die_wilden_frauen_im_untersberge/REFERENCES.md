# The Wild Women of the Untersberg: Reference

## Line of Work

Staging _Die wilden Frauen im Untersberge_, legend no. 50 of the Brothers Grimm's _Deutsche Sagen_. Wild women emerge from the Untersberg to feed shepherd children and assist with the harvest before returning.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 50.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 50 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
