# The False Oath: Reference

## Line of Work

Staging _Der falsche Eid_, legend no. 101 of the Brothers Grimm's _Deutsche Sagen_. A peasant swears a false oath to claim land, only for the earth to open and swallow him up.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 101.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 101 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
