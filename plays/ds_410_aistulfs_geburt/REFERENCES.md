# Aistulf's Birth: Reference

## Line of Work

Staging _Aistulfs Geburt_, legend no. 410 of the Brothers Grimm's _Deutsche Sagen_. A king father tests five newborn sons with a spear, sparing and naming the child who grabs the shaft.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 410.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 410 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
