# Freiherr Albert of Simmern: Reference

## Line of Work

Staging _Freiherr Albert von Simmern_, legend no. 534 of the Brothers Grimm's _Deutsche Sagen_. Albert hunts a white stag that leads him to an underground fairy palace.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 534.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 534 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
