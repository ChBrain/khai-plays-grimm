# The Changeling: Reference

## Line of Work

Staging _Der Wechselbalg_, legend no. 82 of the Brothers Grimm's _Deutsche Sagen_. A mother discovers her crying, insatiable child is a changeling when a voice from a bridge calls out to it.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 82.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 82 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
