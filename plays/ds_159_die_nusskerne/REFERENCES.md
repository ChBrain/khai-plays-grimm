# The Nut Kernels: Reference

## Line of Work

Staging _Die Nußkerne_, legend no. 159 of the Brothers Grimm's _Deutsche Sagen_. A lazy boy following a mysterious nut-cracking lady discovers an underground cellar filled with barrels of gold.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 159.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 159 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
