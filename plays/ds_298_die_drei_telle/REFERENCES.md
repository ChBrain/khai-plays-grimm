# The Three Tells: Reference

## Line of Work

Staging _Die drei Telle_, legend no. 298 of the Brothers Grimm's _Deutsche Sagen_. Three legendary Swiss liberators sleep in a hidden cave, briefly waking to ask a shepherd if the time has come.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 298.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 298 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
