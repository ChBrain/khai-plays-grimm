# The Maiden's Leap: Reference

## Line of Work

Staging _Jungfrausprung_, legend no. 142 of the Brothers Grimm's _Deutsche Sagen_. A pure country girl leaps across the Mur river from a mountain to escape a wicked pursuer, naming the cliff.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 142.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 142 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
