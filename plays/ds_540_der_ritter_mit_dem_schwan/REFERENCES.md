# The Knight of the Swan: Reference

## Line of Work

Staging _Der Ritter mit dem Schwan_, legend no. 540 of the Brothers Grimm's _Deutsche Sagen_. Beatrix bears septuplets with silver chains; Matabruna steals them, turning the boys into swans.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 540.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 540 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
