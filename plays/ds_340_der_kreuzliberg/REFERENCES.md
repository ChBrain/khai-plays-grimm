# The Kreuzliberg: Reference

## Line of Work

Staging _Der Kreuzliberg_, legend no. 340 of the Brothers Grimm's _Deutsche Sagen_. A princess is turned to stone by mountain spirits but is later redeemed by a holy image.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 340.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 340 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
