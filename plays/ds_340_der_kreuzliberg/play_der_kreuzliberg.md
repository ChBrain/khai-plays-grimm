# Play: Der Kreuzliberg

## Estate

[khai-plays-grimm](https://github.com/ChBrain/khai-plays-grimm): the repository that holds this production and answers for it.

## Name

Der Kreuzliberg.

## Arc

Eine Königstochter steigt nachts in eine durch einen Sturm aufgerissene Felsspalte hinab, wird dort von wilden Geistern ergriffen und in den Abgrund gezogen. Am folgenden Tag findet man sie versteinert in Baumgestalt auf einer Anhöhe. Erst durch die Ankunft eines klösterlichen Wunderbildes wird sie von dem Fluch erlöst. Der Berg wird daraufhin mit einem Kreuz geweiht.
