# The Ancestress of Rantzau: Reference

## Line of Work

Staging _Die Ahnfrau von Rantzau_, legend no. 41 of the Brothers Grimm's _Deutsche Sagen_. A dwarf leads the grandmother of Rantzau to help a Zwergin give birth, rewarding her with gold under strict warnings.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 41.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 41 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
