# The Gushing Silver: Reference

## Line of Work

Staging _Das quellende Silber_, legend no. 161 of the Brothers Grimm's _Deutsche Sagen_. A poor girl meeting a white-dressed man in the forest receives a basket of silver coins gushing from a hill.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 161.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 161 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
