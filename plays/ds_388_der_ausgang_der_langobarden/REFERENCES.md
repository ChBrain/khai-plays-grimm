# The Departure of the Lombards: Reference

## Line of Work

Staging _Der Ausgang der Langobarden_, legend no. 388 of the Brothers Grimm's _Deutsche Sagen_. Overpopulated Winili draw lots to migrate under Ibor and Ayo, defeating Vandals and settling in Italy.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 388.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 388 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
