# The Trial of the Fox: Reference

## Line of Work

Staging _The Trial of the Fox_, an episode from Jacob Grimm's _Reinhart Fuchs_. Reinhart is tried, found guilty, and sentenced to hang on the gallows.

## Origin

Jacob Grimm's edition of the animal epic _Reinhart Fuchs_ (1834).

| Source          | Key Work / Event        | Scope                                           |
| --------------- | ----------------------- | ----------------------------------------------- |
| **Jacob Grimm** | _Reinhart Fuchs_ (1834) | The trickster episode as Jacob Grimm edited it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider epic**: the production stages this episode alone, not the full beast-epic it belongs to.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
