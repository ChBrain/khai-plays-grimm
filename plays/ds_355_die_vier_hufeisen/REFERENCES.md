# The Four Horseshoes: Reference

## Line of Work

Staging _Die vier Hufeisen_, legend no. 355 of the Brothers Grimm's _Deutsche Sagen_. Count Ernst of Klettenberg rides his horse into a church during vespers, causing the horseshoes to fall off.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 355.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 355 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
