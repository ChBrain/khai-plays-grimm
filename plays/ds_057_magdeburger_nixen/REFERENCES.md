# The Magdeburg Nixes: Reference

## Line of Work

Staging _Magdeburger Nixen_, legend no. 57 of the Brothers Grimm's _Deutsche Sagen_. A nix drags a skilled swimmer into the depths of the Elbe and later thwarts the construction of a water pipe.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 57.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 57 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
