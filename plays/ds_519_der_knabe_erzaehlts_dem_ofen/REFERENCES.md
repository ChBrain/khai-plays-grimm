# The Boy Tells the Oven: Reference

## Line of Work

Staging _Der Knabe erzählts dem Ofen_, legend no. 519 of the Brothers Grimm's _Deutsche Sagen_. A boy bound by oath reveals a coup plot by telling it to a tiled stove in a butcher's guildhall.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 519.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 519 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
