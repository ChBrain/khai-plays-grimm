# Winkelried and the Dragon: Reference

## Line of Work

Staging _Winkelried und der Lindwurm_, legend no. 218 of the Brothers Grimm's _Deutsche Sagen_. An exiled hero kills a devastating dragon using thorns but dies from the monster's toxic blood on his skin.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 218.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 218 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
