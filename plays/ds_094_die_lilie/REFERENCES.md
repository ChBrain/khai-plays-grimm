# The Lily: Reference

## Line of Work

Staging _Die Lilie_, legend no. 94 of the Brothers Grimm's _Deutsche Sagen_. A nobleman cuts off a servant's head at a banquet but cannot put it back until he kills a hidden disruptor.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 94.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 94 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
