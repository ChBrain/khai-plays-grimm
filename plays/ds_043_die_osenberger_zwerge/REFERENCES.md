# The Osenberg Dwarfs: Reference

## Line of Work

Staging _Die Osenberger Zwerge_, legend no. 43 of the Brothers Grimm's _Deutsche Sagen_. Osenberg dwarfs pay for beer with silver; a forgotten jug brings luck to the family until it breaks.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 43.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 43 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
