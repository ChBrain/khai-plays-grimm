# The Ox Hill: Reference

## Line of Work

Staging _Der Ochsenberg_, legend no. 113 of the Brothers Grimm's _Deutsche Sagen_. A noblewoman warned in a dream flees a sinking castle on an ox, watching the ruins sink to form a lake.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 113.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 113 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
