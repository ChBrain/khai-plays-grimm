# Hermann von Treffurt: Reference

## Line of Work

Staging _Hermann von Treffurt_, legend no. 580 of the Brothers Grimm's _Deutsche Sagen_. A robber knight misses his path in the dark and is saved by his horse, leading him to repent and become a monk.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 580.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 580 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
