# The Maiden of Staufenberg: Reference

## Line of Work

Staging _Das Fräulein vom Staufenberg_, legend no. 228 of the Brothers Grimm's _Deutsche Sagen_. An enchanted castle maiden with golden curly hair periodically appears on a Harz mountain cliff bearing her footprint.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 228.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 228 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
