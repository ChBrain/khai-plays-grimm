# Albertus Magnus and Emperor William: Reference

## Line of Work

Staging _Albertus Magnus und Kaiser Wilhelm_, legend no. 495 of the Brothers Grimm's _Deutsche Sagen_. Albertus Magnus hosts a winter garden banquet where he temporarily conjures a warm, blooming summer.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 495.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 495 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
