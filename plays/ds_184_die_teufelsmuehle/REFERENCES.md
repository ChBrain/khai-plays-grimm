# The Devil's Mill: Reference

## Line of Work

Staging _Die Teufelsmühle_, legend no. 184 of the Brothers Grimm's _Deutsche Sagen_. A miller makes a pact with the devil to build a windmill, but a premature rooster crow causes its destruction.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 184.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 184 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
