# The Well of Robbedissen: Reference

## Line of Work

Staging _Der Robbedisser Brunnen_, legend no. 294 of the Brothers Grimm's _Deutsche Sagen_. A spring, a black ground, and a giant poplar tree warn local villagers of an impending bloody battle in the field.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 294.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 294 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
