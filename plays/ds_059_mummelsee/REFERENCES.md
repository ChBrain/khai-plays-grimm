# Mummelsee: Reference

## Line of Work

Staging _Mummelsee_, legend no. 59 of the Brothers Grimm's _Deutsche Sagen_. Throwing stones into the bottomless Mummelsee triggers severe storms until water sprites return the stones.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 59.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 59 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
