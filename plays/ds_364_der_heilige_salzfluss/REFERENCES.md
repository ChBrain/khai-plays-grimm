# The Holy Salt River: Reference

## Line of Work

Staging _Der heilige Salzfluss_, legend no. 364 of the Brothers Grimm's _Deutsche Sagen_. Germanic tribes fight for a salt-rich river, believing its divine harvest connects them directly to the gods.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 364.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 364 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
