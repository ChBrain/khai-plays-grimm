# King Liutprand's Feet: Reference

## Line of Work

Staging _König Liutprands Füße_, legend no. 409 of the Brothers Grimm's _Deutsche Sagen_. King Liutprand's exceptionally long feet become the standard unit for measuring Goth fields.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 409.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 409 | The legend as the Grimms set it down. |

What the production refuses to model, and to whom it delegates.
