# The Christ Image at Wittenberg: Reference

## Line of Work

Staging _Das Christus-Bild zu Wittenberg_, legend no. 347 of the Brothers Grimm's _Deutsche Sagen_. A miraculous image of Christ in Wittenberg always appears exactly one inch taller than the viewer.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 347.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 347 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
