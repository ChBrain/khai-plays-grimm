# The Oxen at Melchtal: Reference

## Line of Work

Staging _Die Ochsen auf dem Acker zu Melchtal_, legend no. 515 of the Brothers Grimm's _Deutsche Sagen_. Bailiff Landenberg tries to seize a peasant's oxen; the son resists, and the father is blinded.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 515.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 515 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.
