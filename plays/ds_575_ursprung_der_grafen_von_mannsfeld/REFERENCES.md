# Origin of the Counts of Mansfeld: Reference

## Line of Work

Staging _Ursprung der Grafen von Mannsfeld_, legend no. 575 of the Brothers Grimm's _Deutsche Sagen_. A knight clever sows a bushel of barley in a thin border to claim the entire County of Mansfeld.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 575.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 575 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
