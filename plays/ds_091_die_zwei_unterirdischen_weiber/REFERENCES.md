# The Two Underground Women: Reference

## Line of Work

Staging _Die zwei unterirdischen Weiber_, legend no. 91 of the Brothers Grimm's _Deutsche Sagen_. Two underground women swap a sleeping newborn, but a quick-witted maid saves the child by a clever counter-swap.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 91.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 91 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
