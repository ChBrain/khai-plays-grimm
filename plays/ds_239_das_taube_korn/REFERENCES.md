# The Deaf Grain: Reference

## Line of Work

Staging _Das taube Korn_, legend no. 239 of the Brothers Grimm's _Deutsche Sagen_. A rich widow who dumps grain into the sea has her harbor blocked by a sandbank where only deaf grass grows.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 239.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 239 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
