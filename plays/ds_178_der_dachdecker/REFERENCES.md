# The Roofer: Reference

## Line of Work

Staging _Der Dachdecker_, legend no. 178 of the Brothers Grimm's _Deutsche Sagen_. A young roofer speaks irrationally from a tower top before falling to his death while his father prays for him.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 178.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 178 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
