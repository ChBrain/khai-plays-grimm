# Devil's Ditch: Reference

## Line of Work

Staging _Teufels-Graben_, legend no. 339 of the Brothers Grimm's _Deutsche Sagen_. A farmer accepts the devil's help to dig a ditch, but the seven helper workers pay with their lives.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 339.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 339 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
