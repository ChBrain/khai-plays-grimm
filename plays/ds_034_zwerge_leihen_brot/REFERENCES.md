# Dwarfs Borrow Bread: Reference

## Line of Work

Staging _Zwerge leihen Brot_, legend no. 34 of the Brothers Grimm's _Deutsche Sagen_. A dwarf woman borrows fresh bread from a farmer, returning a warm loaf, but warns that human sins drive them away.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 34.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 34 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
