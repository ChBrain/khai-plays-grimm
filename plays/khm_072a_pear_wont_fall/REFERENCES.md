# References: Das Birnli will nit fallen

Staged from the early/withdrawn Grimm fairy tale _Das Birnli will nit fallen_ (KHM 72a, first edition 1812).

## Source

- **German Title**: Das Birnli will nit fallen
- **KHM Number**: 72a
- **Edition**: First Edition (1812), Volume 1, Number 72
- **Primary Source**: [de.wikisource: Das Birnli will nit fallen](<https://de.wikisource.org/wiki/Das_Birnli_will_nit_fallen_(1812)>)
- **German Anchor Line**: _"Der Herr will das Birnli schüttle, das Birnli will nit fallen: der Herr, der schickt das Jockli hinaus, es soll das Birnli schüttle..."_
