# The Meadow Maiden: Reference

## Line of Work

Staging _Die Wiesenjungfrau_, legend no. 224 of the Brothers Grimm's _Deutsche Sagen_. A fearful shepherd boy thrice refuses to rescue a white meadow maiden who is cursed to wait for a cherrywood cradle.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 224.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 224 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
