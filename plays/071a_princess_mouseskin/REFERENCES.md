# References: Prinzessin Mäusehaut

Staged from the early/withdrawn Grimm fairy tale _Prinzessin Mäusehaut_ (KHM 71a, first edition 1812).

## Source

- **German Title**: Prinzessin Mäusehaut
- **KHM Number**: 71a
- **Edition**: First Edition (1812), Volume 1, Number 71
- **Primary Source**: [de.wikisource: Prinzessin Mäusehaut](<https://de.wikisource.org/wiki/Prinzessin_M%C3%A4usehaut_(1812)>)
