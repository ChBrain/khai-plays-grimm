# The Shepherd Boys: Reference

## Line of Work

Staging _Die Hirtenjungen_, legend no. 158 of the Brothers Grimm's _Deutsche Sagen_. Two shepherd boys find a hoard of coins inside a mountain cave but barely escape a magically closing stone door.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 158.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 158 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
