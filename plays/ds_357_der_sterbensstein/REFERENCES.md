# The Stone of Dying: Reference

## Line of Work

Staging _Der Sterbensstein_, legend no. 357 of the Brothers Grimm's _Deutsche Sagen_. A victim of a murder in the Kirchetbuel gorge leaves a permanent handprint pressed into a solid stone.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 357.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 357 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
