# The Hoselberg: Reference

## Line of Work

Staging _Der Hoselberg_, legend no. 174 of the Brothers Grimm's _Deutsche Sagen_. A Thuringian mountain contains purgatory, prompting an English queen to build a chapel at its base to pray.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 174.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 174 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
