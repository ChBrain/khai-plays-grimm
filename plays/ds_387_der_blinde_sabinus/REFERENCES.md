# The Blind Sabinus: Reference

## Line of Work

Staging _Der blinde Sabinus_, legend no. 387 of the Brothers Grimm's _Deutsche Sagen_. The blind Bishop Sabinus detects Totila's identity by touch and later survives a poison attempt by his archdeacon.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 387.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 387 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
