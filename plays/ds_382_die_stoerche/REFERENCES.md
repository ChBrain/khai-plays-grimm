# The Storks: Reference

## Line of Work

Staging _Die Störche_, legend no. 382 of the Brothers Grimm's _Deutsche Sagen_. Attila interprets storks leaving a besieged city as a prophetic sign, motivating his army to sack Aquileia.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 382.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 382 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
