# Dwarf Mountains: Reference

## Line of Work

Staging _Zwergberge_, legend no. 33 of the Brothers Grimm's _Deutsche Sagen_. Near Aachen, Jena, and Hohenstein, dwarfs borrow kitchen utensils from humans for their weddings and return them.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 33.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 33 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
