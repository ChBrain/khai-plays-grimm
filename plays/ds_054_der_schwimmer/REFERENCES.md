# The Swimmer: Reference

## Line of Work

Staging _Der Schwimmer_, legend no. 54 of the Brothers Grimm's _Deutsche Sagen_. A boastful baker's apprentice bets he can swim across the Elbe three times, only to be dragged down by a Nix.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 54.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 54 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
