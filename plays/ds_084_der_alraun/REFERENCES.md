# The Mandrake: Reference

## Line of Work

Staging _Der Alraun_, legend no. 84 of the Brothers Grimm's _Deutsche Sagen_. A mandrake root, grown under a gallows, is dug up using a black dog to avoid its fatal shriek, bringing wealth.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 84.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 84 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
