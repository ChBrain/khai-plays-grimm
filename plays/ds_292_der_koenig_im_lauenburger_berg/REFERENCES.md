# The King in the Lauenburg Mountain: Reference

## Line of Work

Staging _Der König im lauenburger Berg_, legend no. 292 of the Brothers Grimm's _Deutsche Sagen_. Two condemned criminals descend into a Kashubian abyss and find a hidden garden and a king sitting on a silver chair.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 292.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 292 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
