# Play: Der König im lauenburger Berg

## Estate

[khai-plays-grimm](https://github.com/ChBrain/khai-plays-grimm): the repository that holds this production and answers for it.

## Name

Der König im lauenburger Berg.

## Arc

Im Jahr 1596 tut sich nahe der Lauenburg in Kaschuben eine riesige Kluft auf. Zwei zum Tode verurteilte Verbrecher erhalten die Chance auf Begnadigung, wenn sie in den Abgrund hinabsteigen. In der Tiefe entdecken sie einen wunderschönen Garten mit einem blühenden, unberührbaren Baum. Ein geisterhaftes Kind führt sie über eine weite Ebene zu einem prächtigen Schloss. Drinnen ertönt Saitenmusik, und ein König sitzt auf einem silbernen Thron. Er hält ein Zepter und übergibt den Männern über das Kind einen geheimnisvollen Brief.
