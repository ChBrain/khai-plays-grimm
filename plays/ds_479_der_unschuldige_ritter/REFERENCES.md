# The Innocent Knight: Reference

## Line of Work

Staging _Der unschuldige Ritter_, legend no. 479 of the Brothers Grimm's _Deutsche Sagen_. An innocent knight falsely accused of raping the Empress flows white milk instead of blood at execution.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 479.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 479 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
