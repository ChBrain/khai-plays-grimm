# Hanging in Chains: Reference

## Line of Work

Staging _In Ketten aufhängen_, legend no. 570 of the Brothers Grimm's _Deutsche Sagen_. Philip literally hangs Heinz von Lüder in a golden chain to satisfy a imperial command without hurting him.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 570.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 570 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
