# The Immigration of the Huns: Reference

## Line of Work

Staging _Die Einwanderung der Hunnen_, legend no. 379 of the Brothers Grimm's _Deutsche Sagen_. A miraculous female deer guides Hun hunters through an impassable swamp, revealing a secret path to Scythia.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 379.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 379 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
