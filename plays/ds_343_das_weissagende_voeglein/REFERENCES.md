# The Prophetic Bird: Reference

## Line of Work

Staging _Das weissagende Vöglein_, legend no. 343 of the Brothers Grimm's _Deutsche Sagen_. A tiny white bird on an oak tree gives Barbara Sellentius a prophecy for the Duke of Pomerania.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 343.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 343 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
