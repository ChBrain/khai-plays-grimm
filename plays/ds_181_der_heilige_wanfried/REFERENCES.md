# Saint Winfried: Reference

## Line of Work

Staging _Der heilige Winfried_, legend no. 181 of the Brothers Grimm's _Deutsche Sagen_. Saint Boniface stamps a foot imprint into the Christenberg and causes a dry wooden staff to sprout green leaves.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 181.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 181 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
