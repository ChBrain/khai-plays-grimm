# The Shears and the Sword: Reference

## Line of Work

Staging _Die Scheere und das Schwert_, legend no. 431 of the Brothers Grimm's _Deutsche Sagen_. A mourning Queen chooses death over shearing for her grandsons; the brutal kings murder the boys.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 431.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 431 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
