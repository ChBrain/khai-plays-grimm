# Death of the Firstborn: Reference

## Line of Work

Staging _Tod des Erstgebornen_, legend no. 261 of the Brothers Grimm's _Deutsche Sagen_. An innocent nurse is executed for suffocating a baby and leaves behind a curse on the family's firstborn sons.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 261.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 261 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
