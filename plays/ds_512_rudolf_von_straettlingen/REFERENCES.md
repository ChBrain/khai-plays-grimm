# Rudolf of Strättlingen: Reference

## Line of Work

Staging _Rudolf von Strättlingen_, legend no. 512 of the Brothers Grimm's _Deutsche Sagen_. Saint Michael weighs proud King Rudolf's soul and strikes the Devil to prevent him from pulling down the scale.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 512.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 512 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
