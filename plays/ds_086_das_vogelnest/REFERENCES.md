# The Bird's Nest: Reference

## Line of Work

Staging _Das Vogelnest_, legend no. 86 of the Brothers Grimm's _Deutsche Sagen_. A magical bird's nest is invisible to the eye but visible in water, granting invisibility to its holder.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 86.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 86 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
