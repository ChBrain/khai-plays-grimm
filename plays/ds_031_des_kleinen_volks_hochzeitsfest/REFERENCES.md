# The Little People's Wedding: Reference

## Line of Work

Staging _Des kleinen Volks Hochzeitsfest_, legend no. 31 of the Brothers Grimm's _Deutsche Sagen_. Dwarfs slip into Eilenburg Castle at night to celebrate a wedding, inviting the old Count to join.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 31.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 31 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
