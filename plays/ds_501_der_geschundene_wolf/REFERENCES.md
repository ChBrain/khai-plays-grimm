# The Flayed Wolf: Reference

## Line of Work

Staging _Der geschundene Wolf_, legend no. 501 of the Brothers Grimm's _Deutsche Sagen_. Duke Otto takes Passau and executes papal legate Albert, whose flaying is commemorated in the city coat of arms.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 501.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 501 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
