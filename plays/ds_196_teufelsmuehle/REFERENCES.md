# The Devil's Mill: Reference

## Line of Work

Staging _Teufelsmühle_, legend no. 196 of the Brothers Grimm's _Deutsche Sagen_. The devil builds a Weser windmill and drives the wheel with a rocky stream, creating a local legend.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 196.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 196 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
