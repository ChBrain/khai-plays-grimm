# The Dwarf at the Dance: Reference

## Line of Work

Staging _Das Bergmännlein beim Tanz_, legend no. 39 of the Brothers Grimm's _Deutsche Sagen_. A dwarf from the Wunderberg joins a wedding in Glaß, dances with the maidens, and demands honorable behavior.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 39.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 39 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
