# The Buried Silver Mines: Reference

## Line of Work

Staging _Die verschütteten Silbergruben_, legend no. 98 of the Brothers Grimm's _Deutsche Sagen_. An innocent mine manager cursed a rich mine upon his execution, turning his blood into milk.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 98.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 98 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
