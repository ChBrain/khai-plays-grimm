# The Wild Spirits: Reference

## Line of Work

Staging _Die wilden Geister_, legend no. 151 of the Brothers Grimm's _Deutsche Sagen_. German villagers in Veneto appease the Wild Man and Forest Woman with burnt hair and read spirit footprints in ash.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 151.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 151 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
