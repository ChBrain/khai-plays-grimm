# The Rock Bridge: Reference

## Line of Work

Staging _Die Felsenbrücke_, legend no. 202 of the Brothers Grimm's _Deutsche Sagen_. A shepherd leaps over a heap of coals placed by the devil on a narrow bridge after praying to the Virgin Mary.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 202.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 202 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
