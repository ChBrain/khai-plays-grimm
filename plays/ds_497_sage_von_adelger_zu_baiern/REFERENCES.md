# Legend of Adelger of Bavaria: Reference

## Line of Work

Staging _Sage von Adelger zu Baiern_, legend no. 497 of the Brothers Grimm's _Deutsche Sagen_. Duke Adelger defeats Emperor Severus after his wise adviser warns him via a public stag parable.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 497.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 497 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
