# Legend of Emperor Henry III: Reference

## Line of Work

Staging _Sage von Kaiser Heinrich III._, legend no. 486 of the Brothers Grimm's _Deutsche Sagen_. A voice prophesies a fugitive's newborn son will marry the Emperor's daughter; despite death orders, fate prevails.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 486.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 486 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
