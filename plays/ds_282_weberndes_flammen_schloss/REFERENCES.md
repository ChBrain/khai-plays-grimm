# The Hovering Castle of Flames: Reference

## Line of Work

Staging _Weberndes Flammen-Schloß_, legend no. 282 of the Brothers Grimm's _Deutsche Sagen_. A brave Tyrolean nobleman climbs a cursed mountain to explore a fiery castle and is abducted by a black rider.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 282.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 282 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
