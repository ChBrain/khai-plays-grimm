# Plot: Das Bett mit den Erbsen

## Taxonomy

[The Pea Test](play_the_pea_test.md)

## Owner

- Project: khai-plays-grimm

## Cue

Die Königin nimmt das Mädchen auf und beschließt, im Geheimen eine absolute Probe ihrer Empfindsamkeit durchzuführen.

## Action

Die Königin präpariert das Bett im Turmzimmer, indem sie drei Erbsen direkt auf das Gestell legt und sechs weiche Matratzen sowie Eiderdaunendecken darüber schichtet. Das Mädchen wird zur Ruhe hineingeführt und verbringt die gesamte stürmische Nacht schlaflos auf diesem hohen Lager.

## Stage

[de Thormstuv](place_tower_chamber.md)

## Tension

Die unsichtbaren Erbsen unter den dicken Schichten weicher Polster stellen eine unmerkliche Barriere dar, die nur von äußerster Zartheit wahrgenommen werden kann.
