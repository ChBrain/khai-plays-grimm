# The Holy Springs: Reference

## Line of Work

Staging _Die heiligen Quellen_, legend no. 103 of the Brothers Grimm's _Deutsche Sagen_. Three holy springs sprout on Rütli meadow after the Swiss oath, while a traitor burns from within.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 103.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 103 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
