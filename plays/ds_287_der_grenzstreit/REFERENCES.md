# The Boundary Dispute: Reference

## Line of Work

Staging _Der Grenzstreit_, legend no. 287 of the Brothers Grimm's _Deutsche Sagen_. Villagers resolve a land border dispute by tracking a crawfish walking across a field to place their boundary stones.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 287.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 287 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
