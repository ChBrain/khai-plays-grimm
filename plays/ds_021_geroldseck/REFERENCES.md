# Geroldseck: Reference

## Line of Work

Staging _Geroldseck_, legend no. 21 of the Brothers Grimm's _Deutsche Sagen_. Ancient German heroes rest inside Castle Geroldseck, waiting to emerge and save their people in times of need.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 21.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 21 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
