# Siegfried and Genevieve: Reference

## Line of Work

Staging _Siegfried und Genovefa_, legend no. 538 of the Brothers Grimm's _Deutsche Sagen_. Genovefa is falsely accused of adultery by Golo and cast out, surviving in a cave with a doe.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 538.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 538 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
