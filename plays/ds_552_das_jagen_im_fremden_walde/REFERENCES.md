# Hunting in the Foreign Forest: Reference

## Line of Work

Staging _Das Jagen im fremden Walde_, legend no. 552 of the Brothers Grimm's _Deutsche Sagen_. Adelheid plots with Ludwig of Thuringia to murder her husband Pfalzgraf Friedrich during an illegal hunt.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 552.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 552 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
