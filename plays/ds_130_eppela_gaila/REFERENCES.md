# Eppela Gaila: Reference

## Line of Work

Staging _Eppela Gaila_, legend no. 130 of the Brothers Grimm's _Deutsche Sagen_. The robber knight Eppela Gaila Gailingen uses his magical horse to escape from the Nuremberg forces.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 130.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 130 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
