# Giant Einheer: Reference

## Line of Work

Staging _Riese Einheer_, legend no. 18 of the Brothers Grimm's _Deutsche Sagen_. The giant warrior Einheer wades through rivers, pulls his horse by the tail, and fights like an entire army.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 18.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 18 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
