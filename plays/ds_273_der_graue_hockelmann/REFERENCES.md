# The Gray Hockelmann: Reference

## Line of Work

Staging _Der graue Hockelmann_, legend no. 273 of the Brothers Grimm's _Deutsche Sagen_. A grey ghost forces an Auerbach farmer to carry him up to the castle hill, leading to the farmer's fatal exhaustion.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 273.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 273 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
