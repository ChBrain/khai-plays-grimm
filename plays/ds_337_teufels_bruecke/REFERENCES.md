# Devil's Bridge: Reference

## Line of Work

Staging _Teufels-Brücke_, legend no. 337 of the Brothers Grimm's _Deutsche Sagen_. A shepherd tricks the devil into building a bridge across the Reuss by sending a chamois across it first.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 337.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 337 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
