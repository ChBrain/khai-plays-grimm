# Sivard's Death: Reference

## Line of Work

Staging _Sivard's Death_, a ballad from Wilhelm Grimm's _Altdänische Heldenlieder_. Sivard rides his horse over the glass mountain, but falls victim to Queen Brynhild's murderous plot.

## Origin

Wilhelm Grimm's translation of _Altdänische Heldenlieder, Balladen und Märchen_ (1811).

| Source            | Key Work / Event                  | Scope                                           |
| ----------------- | --------------------------------- | ----------------------------------------------- |
| **Wilhelm Grimm** | _Altdänische Heldenlieder_ (1811) | The Nordic ballad as Wilhelm Grimm rendered it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider tradition**: the production stages this ballad alone, not the Danish cycle it is drawn from.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
