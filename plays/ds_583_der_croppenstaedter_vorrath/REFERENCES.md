# The Kroppenstedt Store: Reference

## Line of Work

Staging _Der Croppenstädter Vorrath_, legend no. 583 of the Brothers Grimm's _Deutsche Sagen_. A poor cow-herd is blessed with fourteen sons born in a single year, requiring a large baptismal basin.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 583.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 583 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
