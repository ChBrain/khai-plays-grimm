# The Lady of Alvensleben: Reference

## Line of Work

Staging _Die Frau von Alvensleben_, legend no. 68 of the Brothers Grimm's _Deutsche Sagen_. A noblewoman assists a nix-wife giving birth inside a mountain, receiving a golden ring that seals her lineage's fate.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 68.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 68 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
