# The Good Gerhard Swan: Reference

## Line of Work

Staging _Der gute Gerhard Schwan_, legend no. 545 of the Brothers Grimm's _Deutsche Sagen_. A silent knight arrives in a swan boat with a letter; he is raised to Duke by Karl and marries Adalis.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 545.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 545 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
