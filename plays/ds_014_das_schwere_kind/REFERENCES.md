# The Heavy Child: Reference

## Line of Work

Staging _Das schwere Kind_, legend no. 14 of the Brothers Grimm's _Deutsche Sagen_. Two noblemen find a child in a bush that cannot be lifted, which prophesies a fruitful but deadly year.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 14.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 14 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
