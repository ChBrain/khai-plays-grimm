# The Pear Tree on Walserfeld: Reference

## Line of Work

Staging _Der Birnbaum auf dem Walserfeld_, legend no. 24 of the Brothers Grimm's _Deutsche Sagen_. A dry pear tree on the Walserfeld predicts a terrible final battle when it greens and bears fruit.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 24.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 24 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
