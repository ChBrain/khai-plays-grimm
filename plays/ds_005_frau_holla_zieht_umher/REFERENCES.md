# Mother Holle Wanders: Reference

## Line of Work

Staging _Frau Holla zieht umher_, legend no. 5 of the Brothers Grimm's _Deutsche Sagen_. During the twelve nights of Christmas, Mother Holle wanders to reward industrious spinners and punish the lazy.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 5.

| Source             | Key Work / Event       | Scope                                 |
| ------------------ | ---------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 5 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
