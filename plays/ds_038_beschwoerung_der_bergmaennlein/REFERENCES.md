# Conjuring the Dwarfs: Reference

## Line of Work

Staging _Beschwörung der Bergmännlein_, legend no. 38 of the Brothers Grimm's _Deutsche Sagen_. Paul Creuz conjures dwarfs using milk, honey, and a black hen, eventually calling their King for a book of wisdom.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 38.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 38 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
