# The Paradise of the Animals: Reference

## Line of Work

Staging _Das Paradies der Thiere_, legend no. 301 of the Brothers Grimm's _Deutsche Sagen_. A hidden valley high on the Mattenberg serves as a sanctuary for animals, accessible only once every twenty years.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 301.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 301 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
