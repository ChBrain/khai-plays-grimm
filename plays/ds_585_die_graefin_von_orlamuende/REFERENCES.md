# The Countess of Orlamünde: Reference

## Line of Work

Staging _Die Gräfin von Orlamünde_, legend no. 585 of the Brothers Grimm's _Deutsche Sagen_. Agnes murders her two children to marry Albert, but is rejected and becomes the death-warning White Lady.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 585.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 585 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
