# Fridigern: Reference

## Line of Work

Staging _Fridigern_, legend no. 372 of the Brothers Grimm's _Deutsche Sagen_. Famine-stricken Goths sell their children for dog meat to Roman exploiters, leading to a historic rebellion.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 372.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 372 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
