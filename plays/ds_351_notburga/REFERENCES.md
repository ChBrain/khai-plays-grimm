# Notburga: Reference

## Line of Work

Staging _Notburga_, legend no. 351 of the Brothers Grimm's _Deutsche Sagen_. Princess Notburga escapes a forced marriage by riding a white stag across the Neckar to live in a cave.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 351.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 351 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
