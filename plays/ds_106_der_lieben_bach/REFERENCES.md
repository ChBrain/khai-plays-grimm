# The Stream of Love: Reference

## Line of Work

Staging _Der Lieben-Bach_, legend no. 106 of the Brothers Grimm's _Deutsche Sagen_. Two lovers dig for forty years to bring a spring to Spangenberg, dying in each other's arms as it succeeds.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 106.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 106 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
