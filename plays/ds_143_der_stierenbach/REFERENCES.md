# The Bull Brook: Reference

## Line of Work

Staging _Der Stierenbach_, legend no. 143 of the Brothers Grimm's _Deutsche Sagen_. A shepherd's baptized lamb turns into a terrifying specter that is eventually defeated by a giant milk-fed bull.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 143.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 143 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
