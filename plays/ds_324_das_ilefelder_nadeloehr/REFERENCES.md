# The Ilfeld Needle-Eye: Reference

## Line of Work

Staging _Das ilefelder Nadelöhr_, legend no. 324 of the Brothers Grimm's _Deutsche Sagen_. A giant drops a stone that pinched his shoe, which becomes a narrow tunnel used for stableboy initiation rites.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 324.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 324 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
