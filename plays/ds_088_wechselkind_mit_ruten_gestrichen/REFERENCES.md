# The Changeling Beaten with Rods: Reference

## Line of Work

Staging _Wechselkind mit Ruten gestrichen_, legend no. 88 of the Brothers Grimm's _Deutsche Sagen_. A mother beats a changeling on a Breslau meadow to force the devil to return her real child.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 88.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 88 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
