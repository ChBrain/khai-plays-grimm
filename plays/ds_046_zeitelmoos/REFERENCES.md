# Zeitelmoos: Reference

## Line of Work

Staging _Zeitelmoos_, legend no. 46 of the Brothers Grimm's _Deutsche Sagen_. A rider meets two children laughing in the Zeitelmoos forest; they reappear ahead of him, laughing mockingly.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 46.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 46 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
