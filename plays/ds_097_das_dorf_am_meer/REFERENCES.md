# The Village by the Sea: Reference

## Line of Work

Staging _Das Dorf am Meer_, legend no. 97 of the Brothers Grimm's _Deutsche Sagen_. A proud village mocking a praying woman is buried in sand by oxen digging and a violent storm.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 97.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 97 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
