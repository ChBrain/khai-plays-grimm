# Maiden Ilse: Reference

## Line of Work

Staging _Jungfrau Ilse_, legend no. 317 of the Brothers Grimm's _Deutsche Sagen_. A cursed princess of Ilsenstein gives a charcoal burner a satchel of gold disguised as acorns and pinecones.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 317.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 317 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
