# The Nix at the Kelle: Reference

## Line of Work

Staging _Der Nix an der Kelle_, legend no. 305 of the Brothers Grimm's _Deutsche Sagen_. A midwife delivers a water sprite's baby in a lake cave but later sees a bloody stream representing the child's murder.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 305.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 305 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
