# The Mountain Spirit: Reference

## Line of Work

Staging _Der Berggeist_, legend no. 2 of the Brothers Grimm's _Deutsche Sagen_. The mountain spirit Meister Hämmerling appears in various forms in the silver mines to punish or reward miners.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 2.

| Source             | Key Work / Event       | Scope                                 |
| ------------------ | ---------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 2 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
