# Harras's Leap: Reference

## Line of Work

Staging _Der Harrassprung_, legend no. 322 of the Brothers Grimm's _Deutsche Sagen_. A knight leaps off a steep cliff into the Zschopau valley to escape enemies, surviving while his horse is killed.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 322.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 322 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
