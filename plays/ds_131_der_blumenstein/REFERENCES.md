# The Blumenstein: Reference

## Line of Work

Staging _Der Blumenstein_, legend no. 131 of the Brothers Grimm's _Deutsche Sagen_. A brave village girl wins a midnight bet at a ruined castle and accidentally retrieves a robber's stolen treasure.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 131.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 131 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
