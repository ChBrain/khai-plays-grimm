# Reinstein: Reference

## Line of Work

Staging _Reinstein_, legend no. 110 of the Brothers Grimm's _Deutsche Sagen_. Beneath a Harz castle, stones taken from a cave always return, while sounds of bells and blacksmiths echo.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 110.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 110 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
