# The Sleeping Soldier: Reference

## Line of Work

Staging _Der schlafende Landsknecht_, legend no. 461 of the Brothers Grimm's _Deutsche Sagen_. A soldier's weasel-like soul crosses a stream over a guard's sword, finding gold in a hill.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 461.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 461 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
