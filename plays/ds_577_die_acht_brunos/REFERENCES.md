# The Eight Brunos: Reference

## Line of Work

Staging _Die acht Brunos_, legend no. 577 of the Brothers Grimm's _Deutsche Sagen_. Count Gebhard rescues his eight infant sons whom his fearful wife ordered to be drowned as puppies.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 577.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 577 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
