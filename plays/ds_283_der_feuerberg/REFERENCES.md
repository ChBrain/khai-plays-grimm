# The Fire Mountain: Reference

## Line of Work

Staging _Der Feuerberg_, legend no. 283 of the Brothers Grimm's _Deutsche Sagen_. A guide leads a shepherd into the fiery Feuerberg where a deceased debtor count hands him a proof of his damnation.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 283.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 283 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
