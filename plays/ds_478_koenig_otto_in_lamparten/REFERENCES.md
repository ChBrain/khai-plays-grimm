# King Otto in Lombardy: Reference

## Line of Work

Staging _König Otto in Lamparten_, legend no. 478 of the Brothers Grimm's _Deutsche Sagen_. King Otto issues leather coins in Mailand and later returns to fulfill his justice promise witnessed by a church.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 478.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 478 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
