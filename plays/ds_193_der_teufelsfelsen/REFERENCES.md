# The Devil's Rock: Reference

## Line of Work

Staging _Der Teufelsfelsen_, legend no. 193 of the Brothers Grimm's _Deutsche Sagen_. Satan tempts Christ from the Cößein rock but excludes two rough villages which remain the devil's own territory.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 193.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 193 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
