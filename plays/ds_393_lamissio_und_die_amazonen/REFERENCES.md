# Lamissio and the Amazons: Reference

## Line of Work

Staging _Lamißio und die Amazonen_, legend no. 393 of the Brothers Grimm's _Deutsche Sagen_. Lamissio wins a swimming duel against an Amazon champion, securing river passage for the Lombards.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 393.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 393 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
