# Ludwig Builds a Wall: Reference

## Line of Work

Staging _Ludwig baut eine Mauer_, legend no. 558 of the Brothers Grimm's _Deutsche Sagen_. Ludwig presents his fully armored knights to the Emperor as a living wall surrounding the castle.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 558.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 558 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
