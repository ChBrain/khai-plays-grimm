# The Devil's Horseshoes: Reference

## Line of Work

Staging _Die Teufels-Hufeisen_, legend no. 208 of the Brothers Grimm's _Deutsche Sagen_. The devil rides a cheating beer seller to a blacksmith who delays the shoeing until the morning cock crows.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 208.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 208 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
