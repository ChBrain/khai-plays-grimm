# Queen Adelaide: Reference

## Line of Work

Staging _Königin Adelheid_, legend no. 466 of the Brothers Grimm's _Deutsche Sagen_. Queen Adelaide's besieged castle forces the enemy to retreat by feeding grain to a wild boar, faking abundance.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 466.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 466 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
