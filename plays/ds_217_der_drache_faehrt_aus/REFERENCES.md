# The Dragon Flies Out: Reference

## Line of Work

Staging _Der Drache fährt aus_, legend no. 217 of the Brothers Grimm's _Deutsche Sagen_. A Swiss cooper survives the winter inside a deep pit with two dragons and escapes by holding onto a dragon's tail.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 217.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 217 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
