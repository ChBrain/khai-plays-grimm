# A Ghostly Rider: Reference

## Line of Work

Staging _Ein gespenstiger Reuter_, legend no. 100 of the Brothers Grimm's _Deutsche Sagen_. An outstanding horse trainer is recognized as a dead soldier, leaves in a jester's outfit, and tricks a groom.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 100.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 100 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
