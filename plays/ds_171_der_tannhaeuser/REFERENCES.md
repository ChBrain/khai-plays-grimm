# Tannhaeuser: Reference

## Line of Work

Staging _Der Tannhäuser_, legend no. 171 of the Brothers Grimm's _Deutsche Sagen_. Knight Tannhäuser leaves Rome in despair when Pope Urban declares his sins unforgiven unless a dry staff sprouts.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 171.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 171 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
