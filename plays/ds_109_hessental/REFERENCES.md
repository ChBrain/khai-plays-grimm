# Hessental: Reference

## Line of Work

Staging _Hessental_, legend no. 109 of the Brothers Grimm's _Deutsche Sagen_. A speaking bird warns a Germanic princess of Roman soldiers marching through the valley with gleaming armor.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 109.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 109 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
