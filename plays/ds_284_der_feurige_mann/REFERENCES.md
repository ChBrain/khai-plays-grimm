# The Fiery Man: Reference

## Line of Work

Staging _Der feurige Mann_, legend no. 284 of the Brothers Grimm's _Deutsche Sagen_. A transparent fiery man walking between boundary stones and spitting fire from his mouth terrifies the local villagers.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 284.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 284 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
