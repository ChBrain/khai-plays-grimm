# The Altenberg Church: Reference

## Line of Work

Staging _Die altenberger Kirche_, legend no. 291 of the Brothers Grimm's _Deutsche Sagen_. Altenberg villagers fail to move a mountaintop church down to the valley as its stones return uphill every night.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 291.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 291 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
