# Rustling in the Giant's Grave: Reference

## Line of Work

Staging _Es rauscht im Hühnen-Grab_, legend no. 327 of the Brothers Grimm's _Deutsche Sagen_. Stonecutters digging up giant graves in Pomerania flee after hearing a spooky rustling of keys around them.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 327.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 327 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
