# Das Hildebrandslied: Reference

## Line of Work

Staging _Das Hildebrandslied_, an episode from Wilhelm Grimm's _Deutsche Heldensage_. The tragic duel between Hildebrand and his son Hadubrand.

## Origin

Wilhelm Grimm's critical compilation _Deutsche Heldensage_ (1829).

| Source            | Key Work / Event             | Scope                                          |
| ----------------- | ---------------------------- | ---------------------------------------------- |
| **Wilhelm Grimm** | _Deutsche Heldensage_ (1829) | The saga episode as Wilhelm Grimm compiled it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider saga**: the production stages this episode alone, not the full heroic cycle it belongs to.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
