# Henneberg: Reference

## Line of Work

Staging _Henneberg_, legend no. 576 of the Brothers Grimm's _Deutsche Sagen_. A noble lord sees a wild birch hen fly up from a hill, naming his new castle Henneberg after it.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 576.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 576 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
