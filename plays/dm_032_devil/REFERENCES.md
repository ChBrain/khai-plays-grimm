# Devil: Reference

## Line of Work

Staging _Devil_, chapter 32 of Jacob Grimm's _Deutsche Mythologie_, as a cosmological ruleset rather than a narrative. The pagan legacy and attributes in the christianized devil.

## Origin

Jacob Grimm's _Deutsche Mythologie_ (1835), chapter 32, in the definitive 4th edition (posthumous, 1875), through James Steven Stallybrass's public-domain translation _Teutonic Mythology_ (1880-1888).

| Source          | Key Work / Event                   | Scope                                                                     |
| --------------- | ---------------------------------- | ------------------------------------------------------------------------- |
| **Jacob Grimm** | _Deutsche Mythologie_, ch. 32      | The scholarly account of the folk-belief the chapter treats.              |
| **Stallybrass** | _Teutonic Mythology_ (translation) | The public-domain English rendering the house's structural labels follow. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Narrative arc**: this is a ruleset, not a tale; the plots model conceptual stages, not events.
- **Doctrine**: folk-belief is modelled as Grimm reports it, taking no position on the beliefs described.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
