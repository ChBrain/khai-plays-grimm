# The Devil's Dance Floor: Reference

## Line of Work

Staging _Des Teufels Tanzplatz_, legend no. 190 of the Brothers Grimm's _Deutsche Sagen_. The devil builds walls to divide the Earth with God and dances in celebration on a rocky plateau in the Harz.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 190.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 190 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
