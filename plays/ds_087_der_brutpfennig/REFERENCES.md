# The Breeding Penny: Reference

## Line of Work

Staging _Der Brutpfenning_, legend no. 87 of the Brothers Grimm's _Deutsche Sagen_. A person gets a breeding penny from the devil on Christmas Eve, which later burns down a house in Pantschdorf.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 87.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 87 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
