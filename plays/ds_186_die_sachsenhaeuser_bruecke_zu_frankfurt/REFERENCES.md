# The Sachsenhausen Bridge in Frankfurt: Reference

## Line of Work

Staging _Die Sachsenhäuser Brücke zu Frankfurt_, legend no. 186 of the Brothers Grimm's _Deutsche Sagen_. A builder tricks the devil by driving a rooster across a newly completed bridge built under a soul contract.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 186.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 186 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
