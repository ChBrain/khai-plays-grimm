# The Wichts: Reference

## Line of Work

Staging _Die Wichtlein_, legend no. 37 of the Brothers Grimm's _Deutsche Sagen_. Tiny dwarfs dressed as miners haunt the pits, teasing workers and knocking three times as an omen of death.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 37.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 37 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
