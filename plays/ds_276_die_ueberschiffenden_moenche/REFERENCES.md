# The Ferrying Monks: Reference

## Line of Work

Staging _Die überschiffenden Mönche_, legend no. 276 of the Brothers Grimm's _Deutsche Sagen_. Ghostly monks cross the Rhine in Speyer, beating a fisherman during a storm before flying away in a fiery cart.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 276.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 276 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
