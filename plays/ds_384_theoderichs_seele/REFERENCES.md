# Theodoric's Soul: Reference

## Line of Work

Staging _Theoderichs Seele_, legend no. 384 of the Brothers Grimm's _Deutsche Sagen_. A holy hermit on Lipari sees a vision of the deceased King Theodoric being thrown into the volcano crater.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 384.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 384 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
