# The Devil's Stone near Reichenbach: Reference

## Line of Work

Staging _Teufelsstein bei Reichenbach_, legend no. 198 of the Brothers Grimm's _Deutsche Sagen_. The devil builds a complex stone shelter in a single night to house local villagers during war times.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 198.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 198 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
