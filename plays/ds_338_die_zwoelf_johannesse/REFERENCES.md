# The Twelve Johns: Reference

## Line of Work

Staging _Die zwölf Johanneße_, legend no. 338 of the Brothers Grimm's _Deutsche Sagen_. The devil takes one of the king's twelve Johns every year as a toll from their flying magic disk.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 338.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 338 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
