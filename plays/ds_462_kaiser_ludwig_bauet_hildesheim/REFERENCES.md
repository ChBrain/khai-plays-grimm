# Emperor Louis Builds Hildesheim: Reference

## Line of Work

Staging _Kaiser Ludwig bauet Hildesheim_, legend no. 462 of the Brothers Grimm's _Deutsche Sagen_. A miracle fixes a Marian image to a tree stump, and a summer snowfall defines the boundaries of Hildesheim Cathedral.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 462.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 462 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
