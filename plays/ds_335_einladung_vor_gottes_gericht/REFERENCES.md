# Invitation Before God's Court: Reference

## Line of Work

Staging _Einladung vor Gottes Gericht_, legend no. 335 of the Brothers Grimm's _Deutsche Sagen_. A nobleman mockingly invites hanged thieves to dinner, leading to a chilling prophecy and his own execution.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 335.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 335 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
