# The Rat Catcher: Reference

## Line of Work

Staging _Der Rattenfänger_, legend no. 246 of the Brothers Grimm's _Deutsche Sagen_. A miller takes a magic cane left by a rat catcher, causing all the rats in the village to follow him to his mill.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 246.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 246 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
