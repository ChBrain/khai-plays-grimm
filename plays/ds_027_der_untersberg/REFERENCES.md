# The Untersberg: Reference

## Line of Work

Staging _Der Untersberg_, legend no. 27 of the Brothers Grimm's _Deutsche Sagen_. The Untersberg near Salzburg contains palaces, churches, and treasures guarded by small midnight-walking dwarfs.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 27.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 27 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
