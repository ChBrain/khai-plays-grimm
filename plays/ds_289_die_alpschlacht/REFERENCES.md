# The Battle of the Alp: Reference

## Line of Work

Staging _Die Alpschlacht_, legend no. 289 of the Brothers Grimm's _Deutsche Sagen_. Entlebuch shepherds silently steal Obwalden cattle by removing their cowbells and win the subsequent armed conflict.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 289.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 289 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
