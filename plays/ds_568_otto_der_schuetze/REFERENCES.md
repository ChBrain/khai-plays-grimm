# Otto the Archer: Reference

## Line of Work

Staging _Otto der Schütze_, legend no. 568 of the Brothers Grimm's _Deutsche Sagen_. Prince Otto flees to Cleves to serve as an archer, later returning to inherit Hesse.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 568.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 568 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
