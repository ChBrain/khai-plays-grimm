# Saint Kunigunde: Reference

## Line of Work

Staging _Die heilige Kunigund_, legend no. 482 of the Brothers Grimm's _Deutsche Sagen_. Empress Kunigunde walks barefoot over seven red-hot iron plowshares to prove her chaste marriage.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 482.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 482 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
