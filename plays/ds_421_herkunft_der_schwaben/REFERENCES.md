# Origin of the Swabians: Reference

## Line of Work

Staging _Herkunft der Schwaben_, legend no. 421 of the Brothers Grimm's _Deutsche Sagen_. The Swabians' ancestors land at Mount Suevo and their Duke Brenno fights Julius Caesar.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 421.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 421 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
