# The Giant Hidde: Reference

## Line of Work

Staging _Der Riese Hidde_, legend no. 323 of the Brothers Grimm's _Deutsche Sagen_. A giant forest warden presents lion cubs as wolves to his duke and is rewarded with lands that become Hitzacker.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 323.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 323 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
