# The Nix Well: Reference

## Line of Work

Staging _Nixenbrunnen_, legend no. 56 of the Brothers Grimm's _Deutsche Sagen_. Nixes appear at the Nix Well near Kirchhain, and another nix is sighted at the Elisabeth mill in Marburg.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 56.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 56 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
