# Carl Ynach, Salvius Brabon and Lady Swan: Reference

## Line of Work

Staging _Carl Ynach, Salvius Brabon und Frau Schwan_, legend no. 539 of the Brothers Grimm's _Deutsche Sagen_. Exiled Carl Ynach flees Rome with Germana, guided down the Rhine by a mysterious swan.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 539.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 539 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
