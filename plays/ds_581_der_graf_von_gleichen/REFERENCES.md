# The Count of Gleichen: Reference

## Line of Work

Staging _Der Graf von Gleichen_, legend no. 581 of the Brothers Grimm's _Deutsche Sagen_. Count Ludwig escapes Turkish slavery with the Sultan's daughter and returns to live in a bigamous marriage.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 581.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 581 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
