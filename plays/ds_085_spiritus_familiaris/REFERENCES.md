# Spiritus Familiaris: Reference

## Line of Work

Staging _Spiritus familiaris_, legend no. 85 of the Brothers Grimm's _Deutsche Sagen_. A soldier owns a spirit in a bottle that brings wealth, but he must sell it cheaper to avoid damnation.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 85.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 85 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
