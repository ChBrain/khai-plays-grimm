# The Devil as Advocate: Reference

## Line of Work

Staging _Der Teufel als Fürsprecher_, legend no. 211 of the Brothers Grimm's _Deutsche Sagen_. The devil disguised in a blue hat defends a cheated mercenary and carries off the swearing lying landlord.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 211.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 211 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
