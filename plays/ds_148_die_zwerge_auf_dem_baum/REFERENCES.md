# The Dwarves on the Tree: Reference

## Line of Work

Staging _Die Zwerge auf dem Baum_, legend no. 148 of the Brothers Grimm's _Deutsche Sagen_. Helpful dwarves who sit on a maple branch during haymaking are betrayed when locals saw it through at night.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 148.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 148 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
