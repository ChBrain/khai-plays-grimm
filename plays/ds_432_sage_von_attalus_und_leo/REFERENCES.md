# Legend of Attalus and Leo: Reference

## Line of Work

Staging _Sage von Attalus und Leo_, legend no. 432 of the Brothers Grimm's _Deutsche Sagen_. The loyal cook Leo gains a Frankish lord's trust to rescue the noble hostage Attalus on horseback.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 432.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 432 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
