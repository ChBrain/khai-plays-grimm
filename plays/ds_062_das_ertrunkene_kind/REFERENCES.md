# The Drowned Child: Reference

## Line of Work

Staging _Das ertrunkene Kind_, legend no. 62 of the Brothers Grimm's _Deutsche Sagen_. A grieving mother collects her drowned child's bones from the lake; when placed on the altar, the child revives.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 62.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 62 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
