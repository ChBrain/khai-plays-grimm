# The Mouse Tower of Bingen: Reference

## Line of Work

Staging _Der Binger Mäusethurm_, legend no. 242 of the Brothers Grimm's _Deutsche Sagen_. A cruel archbishop who burns starving peasants in a barn is chased and eaten alive by mice in a Rhine tower.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 242.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 242 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
