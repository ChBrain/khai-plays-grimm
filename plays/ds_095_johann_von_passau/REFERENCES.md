# Johann of Passau: Reference

## Line of Work

Staging _Johann von Passau_, legend no. 95 of the Brothers Grimm's _Deutsche Sagen_. A widower’s wife returns from the grave to be his housewife, on the condition that he must never curse again.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 95.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 95 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
