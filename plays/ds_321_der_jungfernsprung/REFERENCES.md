# The Maiden's Leap: Reference

## Line of Work

Staging _Der Jungfernsprung_, legend no. 321 of the Brothers Grimm's _Deutsche Sagen_. A maiden flees a lustful monk at Oybin monastery and leaps off a cliff, saved from injury by descending angels.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 321.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 321 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
