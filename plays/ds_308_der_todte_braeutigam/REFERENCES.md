# The Dead Bridegroom: Reference

## Line of Work

Staging _Der todte Bräutigam_, legend no. 308 of the Brothers Grimm's _Deutsche Sagen_. A magician offers to rescue a drowned bridegroom from a Magdeburg nix in exchange for the couple's souls.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 308.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 308 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
