# Origin of the Saxons: Reference

## Line of Work

Staging _Ursprung der Sachsen_, legend no. 413 of the Brothers Grimm's _Deutsche Sagen_. The Saxons and King Aschanes grow out of the Harz rocks, and Saxon maidens grow on forest trees.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 413.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 413 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
