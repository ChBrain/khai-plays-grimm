# The Cradle from the Tree: Reference

## Line of Work

Staging _Die Wiege aus dem Bäumchen_, legend no. 108 of the Brothers Grimm's _Deutsche Sagen_. A treasure in Rauheneck can only be lifted by a child whose cradle is carved from a tree growing on the tower wall.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 108.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 108 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
