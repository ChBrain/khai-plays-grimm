# The Dwarf Holes: Reference

## Line of Work

Staging _Die Zwerglöcher_, legend no. 303 of the Brothers Grimm's _Deutsche Sagen_. Helpful Harz dwarves lend wedding kitchen utensils to local villagers from their rock cave, demanding timely returns.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 303.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 303 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
