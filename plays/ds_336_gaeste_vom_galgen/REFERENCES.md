# Guests from the Gallows: Reference

## Line of Work

Staging _Gäste vom Galgen_, legend no. 336 of the Brothers Grimm's _Deutsche Sagen_. An innkeeper drunkenly invites three hanged men to dine, receives a terrifying visit, and dies three days later.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 336.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 336 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
