# Dream of the Treasure on the Bridge: Reference

## Line of Work

Staging _Traum vom Schatz auf der Brücke_, legend no. 212 of the Brothers Grimm's _Deutsche Sagen_. A dreamer finds a treasure under a tree after a merchant on a bridge mocks dreams and tells him where to dig.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 212.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 212 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
