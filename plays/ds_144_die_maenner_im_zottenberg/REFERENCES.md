# The Men in Zottenberg: Reference

## Line of Work

Staging _Die Männer im Zottenberg_, legend no. 144 of the Brothers Grimm's _Deutsche Sagen_. A man discovers a hidden cave inside Zottenberg mountain where three silent old robbers await Judgment Day.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 144.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 144 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
