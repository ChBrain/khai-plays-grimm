# Henry the Lion: Reference

## Line of Work

Staging _Heinrich der Löwe_, legend no. 526 of the Brothers Grimm's _Deutsche Sagen_. Henry is shipwrecked, makes a pact with a lion, and is carried home on a griffin to stop his wife's wedding.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 526.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 526 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
