# Walther in the Cloister: Reference

## Line of Work

Staging _Walther im Kloster_, legend no. 412 of the Brothers Grimm's _Deutsche Sagen_. Old hero Walter of Aquitaine does penance in a monastery, defending its transports from robbers with a simple stick.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 412.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 412 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
