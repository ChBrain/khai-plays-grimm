# Maiden Eli: Reference

## Line of Work

Staging _Jungfer Eli_, legend no. 122 of the Brothers Grimm's _Deutsche Sagen_. A cruel housekeeper haunts Frekenhorst abbey until priests banish her ghost to the Davert forest.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 122.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 122 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
