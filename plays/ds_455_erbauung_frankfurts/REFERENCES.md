# The Founding of Frankfurt: Reference

## Line of Work

Staging _Erbauung Frankfurts_, legend no. 455 of the Brothers Grimm's _Deutsche Sagen_. A wild doe guides Charlemagne's fleeing army across a hidden ford on the Main river, founding Frankfurt.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 455.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 455 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
