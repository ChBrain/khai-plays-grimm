# The Swiss Stone Pine Nuts: Reference

## Line of Work

Staging _Die Zirbelnüsse_, legend no. 300 of the Brothers Grimm's _Deutsche Sagen_. A master curses the sweet Swiss pine nut trees to grow only on precipitous cliffs, punishing lazy servants.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 300.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 300 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
