# Origin of the Zaehringers: Reference

## Line of Work

Staging _Ursprung der Zähringer_, legend no. 527 of the Brothers Grimm's _Deutsche Sagen_. A charcoal burner finds silver under his pile and buys a duchy from a deposed King.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 527.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 527 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
