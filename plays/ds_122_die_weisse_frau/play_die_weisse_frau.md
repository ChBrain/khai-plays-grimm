# Play: Die weiße Frau

## Estate

[khai-plays-grimm](https://github.com/ChBrain/khai-plays-grimm): the repository that holds this production and answers for it.

## Name

Die weiße Frau.

## Arc

In einem gräflichen Schloss kämmt eine weiße Frau nachts die Pferde im Stall und zündet dazu eine Wachskerze an. In ihrer eigenen unterirdischen Wohnung ist sie völlig blind, doch außerhalb davon kann sie alles klar sehen. Sie wird eines Tages von einem vorwitzigen Stalljungen belauscht, dem sie zur Strafe das Gesicht verdreht.

## Triggers

**[Plot: Die Arbeit](plot_01_die-arbeit.md)**

Die [weiße Frau](persona_frau.md) vollzieht einen nächtlichen [Spuk](process_spuk.md), indem sie die Pferde mit einer [Wachskerze](piece_wachskerze.md) beleuchtet und mit einem [Kamm](piece_kamm.md) striegelt.

**[Plot: Die Blindheit](plot_02_die-blindheit.md)**

In ihrer [unterirdische Wohnung](place_wohnung.md) ist die Frau vollkommen blind, doch im Schloss sieht sie alles.

**[Plot: Die Störung](plot_03_die-stoerung.md)**

Der [vorwitziger Stalljunge](persona_stalljunge.md) beobachtet sie heimlich bei ihrer Arbeit.

**[Plot: Die Bestrafung](plot_04_die-bestrafung.md)**

Die weiße Frau vollzieht die [Bestrafung](process_bestrafung.md) und verdreht dem Jungen zur Strafe den Kopf.

## Stakes

Ob die weiße Frau ihren Segen vom Stall abzieht, nachdem sie gestört wurde.
