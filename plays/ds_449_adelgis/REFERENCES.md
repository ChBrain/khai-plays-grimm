# Adelgis: Reference

## Line of Work

Staging _Adelgis_, legend no. 449 of the Brothers Grimm's _Deutsche Sagen_. The disguised Prince Adelgis splits marrow bones at Charlemagne's table and escapes in a boat.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 449.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 449 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
