# Leupichis Escapes: Reference

## Line of Work

Staging _Leupichis entflieht_, legend no. 407 of the Brothers Grimm's _Deutsche Sagen_. An escaping captive boy is guided by a wolf and saved from starvation in the wilderness by a prophetic dream.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 407.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 407 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
