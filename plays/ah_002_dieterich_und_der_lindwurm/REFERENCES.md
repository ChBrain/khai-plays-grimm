# King Dieterich and the Dragon: Reference

## Line of Work

Staging _King Dieterich and the Dragon_, a ballad from Wilhelm Grimm's _Altdänische Heldenlieder_. King Dieterich encounters a giant dragon in the wild forest and fights it with the help of a loyal lion.

## Origin

Wilhelm Grimm's translation of _Altdänische Heldenlieder, Balladen und Märchen_ (1811).

| Source            | Key Work / Event                  | Scope                                           |
| ----------------- | --------------------------------- | ----------------------------------------------- |
| **Wilhelm Grimm** | _Altdänische Heldenlieder_ (1811) | The Nordic ballad as Wilhelm Grimm rendered it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider tradition**: the production stages this ballad alone, not the Danish cycle it is drawn from.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
