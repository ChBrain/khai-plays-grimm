# Giants' Pillars: Reference

## Line of Work

Staging _Riesensäulen_, legend no. 19 of the Brothers Grimm's _Deutsche Sagen_. Nine stone pillars in the forest near Miltenberg bear the handprints of giants who tried to build a bridge.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 19.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 19 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
