# Frederick of Oldenburg: Reference

## Line of Work

Staging _Friedrich von Oldenburg_, legend no. 548 of the Brothers Grimm's _Deutsche Sagen_. Friedrich volunteers to fight a lion to save his father, using a straw dummy to trick the beast.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 548.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 548 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
