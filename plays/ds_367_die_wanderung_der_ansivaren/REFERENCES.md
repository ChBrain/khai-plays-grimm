# The Migration of the Ampsivarii: Reference

## Line of Work

Staging _Die Wanderung der Ansivaren_, legend no. 367 of the Brothers Grimm's _Deutsche Sagen_. An exiled Germanic tribe seeking a home is repeatedly driven away until all their youth are wiped out.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 367.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 367 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
