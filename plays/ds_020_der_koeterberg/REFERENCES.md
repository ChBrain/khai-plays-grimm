# The Köterberg: Reference

## Line of Work

Staging _Der Köterberg_, legend no. 20 of the Brothers Grimm's _Deutsche Sagen_. A shepherd finds the entrance to gold inside the Köterberg but is chased away by a bloody man.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 20.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 20 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
