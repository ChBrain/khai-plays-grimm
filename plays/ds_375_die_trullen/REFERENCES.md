# The Trulls: Reference

## Line of Work

Staging _Die Trullen_, legend no. 375 of the Brothers Grimm's _Deutsche Sagen_. Starving Goths are mocked by Vandals with the derogatory name Truller after buying overpriced grain.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 375.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 375 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
