# The Emperor and the Snake: Reference

## Line of Work

Staging _Der Kaiser und die Schlange_, legend no. 459 of the Brothers Grimm's _Deutsche Sagen_. Charlemagne executes an invading toad for a snake, receiving a precious gemstone in gratitude.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 459.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 459 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
