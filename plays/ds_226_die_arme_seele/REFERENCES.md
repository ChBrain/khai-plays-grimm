# The Poor Soul: Reference

## Line of Work

Staging _Die arme Seele_, legend no. 226 of the Brothers Grimm's _Deutsche Sagen_. A sneezing soul under a Paderborn bridge tips over any passing carriage whose driver fails to say a blessing.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 226.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 226 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
