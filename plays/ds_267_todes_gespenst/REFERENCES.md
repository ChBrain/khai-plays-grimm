# The Spectre of Death: Reference

## Line of Work

Staging _Todes-Gespenst_, legend no. 267 of the Brothers Grimm's _Deutsche Sagen_. A shapeshifting spectre in Tyrol peers through windows to signal the imminent death of those residing within.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 267.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 267 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
