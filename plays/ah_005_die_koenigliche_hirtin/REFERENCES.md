# The Royal Shepherdess: Reference

## Line of Work

Staging _The Royal Shepherdess_, a ballad from Wilhelm Grimm's _Altdänische Heldenlieder_. A disguised princess works as a shepherdess until a golden ring reveals her true royal identity to the king.

## Origin

Wilhelm Grimm's translation of _Altdänische Heldenlieder, Balladen und Märchen_ (1811).

| Source            | Key Work / Event                  | Scope                                           |
| ----------------- | --------------------------------- | ----------------------------------------------- |
| **Wilhelm Grimm** | _Altdänische Heldenlieder_ (1811) | The Nordic ballad as Wilhelm Grimm rendered it. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **The wider tradition**: the production stages this ballad alone, not the Danish cycle it is drawn from.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
