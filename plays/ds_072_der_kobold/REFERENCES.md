# The Kobold: Reference

## Line of Work

Staging _Der Kobold_, legend no. 72 of the Brothers Grimm's _Deutsche Sagen_. A helpful kobold assists a household with various chores, bringing fortune as long as he is treated well.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 72.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 72 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
