# The Stone of Lies: Reference

## Line of Work

Staging _Der Lügenstein_, legend no. 201 of the Brothers Grimm's _Deutsche Sagen_. The devil agrees to drop a rock next to the Halberstadt Dom in exchange for a promised wine house.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 201.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 201 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
