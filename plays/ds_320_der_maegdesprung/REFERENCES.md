# The Maiden's Leap: Reference

## Line of Work

Staging _Der Mägdesprung_, legend no. 320 of the Brothers Grimm's _Deutsche Sagen_. A giantess scoops up a ploughman and leaps across the Selke Valley, leaving her footprints pressed into the rock.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 320.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 320 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
