# The Giant's Blood: Reference

## Line of Work

Staging _Das Hühnenblut_, legend no. 326 of the Brothers Grimm's _Deutsche Sagen_. A giant jumping over Westeregeln nicks his toe on a church spire, creating a red pool of giant blood.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 326.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 326 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
