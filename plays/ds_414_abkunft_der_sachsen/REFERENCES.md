# Descent of the Saxons: Reference

## Line of Work

Staging _Abkunft der Sachsen_, legend no. 414 of the Brothers Grimm's _Deutsche Sagen_. Alexander's veterans land on the Elbe and betray the resident Thuringians using large Saxe knives.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 414.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 414 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
