# The Changelings in the Water: Reference

## Line of Work

Staging _Die Wechselbälge im Wasser_, legend no. 83 of the Brothers Grimm's _Deutsche Sagen_. A peasant carrying an insatiable changeling in a basket throws it in the river when the creature speaks.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 83.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 83 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
