# Dietrichstein in Carinthia: Reference

## Line of Work

Staging _Dieterichstein in Kärnthen_, legend no. 509 of the Brothers Grimm's _Deutsche Sagen_. The defenders of Dietrichstein Castle escape silently under the cover of a thick evening fog.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 509.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 509 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
