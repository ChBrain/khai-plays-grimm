# The Raederberg: Reference

## Line of Work

Staging _Der Räderberg_, legend no. 279 of the Brothers Grimm's _Deutsche Sagen_. A Nassau butcher enters a ghostly mansion, drinks a third cup from three monks, and dies three days later.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 279.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 279 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
