# The Werewolf: Reference

## Line of Work

Staging _Der Wärwolf_, legend no. 214 of the Brothers Grimm's _Deutsche Sagen_. A logger transforms into a werewolf using a magic belt and devours a foal while his companion secretly watches.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 214.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 214 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
