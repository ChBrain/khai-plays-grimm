# The Rye Aunt: Reference

## Line of Work

Staging _Die Roggen-Muhme_, legend no. 90 of the Brothers Grimm's _Deutsche Sagen_. A nursing mother is forced to bind sheaves in the field; the Roggen-Mutter swaps her child but returns it later.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 90.

| Source             | Key Work / Event        | Scope                                 |
| ------------------ | ----------------------- | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 90 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
