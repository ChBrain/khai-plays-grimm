# The Boundary Run: Reference

## Line of Work

Staging _Der Gränzlauf_, legend no. 288 of the Brothers Grimm's _Deutsche Sagen_. Uri and Glarus runners compete in a boundary race started by rooster crows, ending in a Glarner runner carrying Uri.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 288.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 288 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
