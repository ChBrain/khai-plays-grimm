# The Wheel of Fortune: Reference

## Line of Work

Staging _Das Glücksrad_, legend no. 210 of the Brothers Grimm's _Deutsche Sagen_. Twelve soldiers spin on the devil's wheel of fortune where one is taken and the rest wake up empty-handed.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 210.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 210 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
