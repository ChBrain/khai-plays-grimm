# The Collapsed Bridge: Reference

## Line of Work

Staging _Die eingefallene Brücke_, legend no. 370 of the Brothers Grimm's _Deutsche Sagen_. After a bridge collapses, half of the migrating Goths are isolated in an unreachable swamp where their echoes persist.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 370.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 370 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
