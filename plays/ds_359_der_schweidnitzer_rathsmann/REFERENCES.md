# The Councilor of Schweidnitz: Reference

## Line of Work

Staging _Der schweidnitzer Rathsmann_, legend no. 359 of the Brothers Grimm's _Deutsche Sagen_. A greedy councilor trains a jackdaw to steal coins from the treasury and dies starving on a high tower ledge.

## Origin

The Brothers Grimm's _Deutsche Sagen_ (1816-1818), legend no. 359.

| Source             | Key Work / Event         | Scope                                 |
| ------------------ | ------------------------ | ------------------------------------- |
| **Brothers Grimm** | _Deutsche Sagen_, DS 359 | The legend as the Grimms set it down. |

## Restrictions

What the production refuses to model, and to whom it delegates.

- **Beyond the legend**: the production stages only what the legend records, not the wider history or tradition it touches.
- **Cross-play context**: strict local isolation. Every link stays within this play's directory.
