# khai plays: Grimm

The Grimm production house: a collection of khai plays, each a
production staged with khai, dedicated to Grimm.

This README is the house's **Estate identity**, the production that answers for
the whole run. Every play in `plays/` links this house in its Estate (E); a play
with no Estate is not yet a production.

- **`plays/`** holds the productions, written in khai-playwright mode.
- **how it was built:** raised by khai-stage (the impresario's generator), wired
  to the khai canon and kit, gated, and protected. See `CLAUDE.md`.
